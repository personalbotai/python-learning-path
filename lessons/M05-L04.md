# Encapsulation: Public, Private, and Protected
**ID**: `M05-L04`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Use naming conventions (_protected, __private), understand name mangling.

## Materi

### Konsep Dasar
Python tidak memiliki strict private/protected modifiers seperti Java/C++. Sebagai gantinya, menggunakan naming conventions:

- **Public**: `name` - bisa diakses dari mana saja
- **Protected**: `_name` - convention: "internal use, treat as non-public" (masih bisa diakses, tapi berhati-hati)
- **Private**: `__name` (double underscore) - name mangling: Python mengubah nama menjadi `_ClassName__name` untuk mengurangi akses dari subclass

**Name Mangling**:
```python
class MyClass:
    def __init__(self):
        self.__private = 42  # becomes _MyClass__private
```
Akses dari luar: `obj._MyClass__private` (bisa, tapi tidak disarankan)

### Detail Teknis
- Protected (`_`) adalah convention only; Python tidak enforce
- Private (`__`) hanya name mangling, bukan true privacy; masih bisa diakses dengan mangled name
- `__init__` dan `__str__` adalah special methods (dunder) bukan private; double underscore di awal dan akhir memiliki makna khusus
- Class attributes juga bisa protected/private: `_class_attr` atau `__class_attr`
- `from module import *` tidak meng-import nama yang dimulai dengan underscore (kecuali didefinisikan di `__all__`)
- Gunakan single underscore untuk "internal use" dan double underscore untuk menghindari name collision di subclass

## Latihan

### Soal 1
Buat class `Person` dengan:
- `__init__(self, name, age)` menyimpan `self.name` (public) dan `self._age` (protected)
- Method `get_age()` mengembalikan `self._age`
- Method `set_age(new_age)` yang validasi: jika new_age >= 0, set `self._age = new_age`; else print "Invalid age"
- Method `_is_adult()` (protected) mengembalikan True jika `self._age >= 18`
- Method `is_adult()` (public) yang memanggil `_is_adult()` dan mengembalikan hasilnya

```python
class Person:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    p = Person("Alice", 25)
    print(p.get_age())       # 25
    print(p.is_adult())      # True
    p.set_age(15)
    print(p.is_adult())      # False
    p.set_age(-5)            # print "Invalid age"
```

**Expected Output**:
```
25
True
False
Invalid age
```

### Soal 2
Buat class `SecretKeeper` dengan:
- `__init__(self, secret)` menyimpan `self.__secret` (private)
- Method `reveal()` yang mengembalikan `self.__secret`
- Method `change_secret(new_secret)` yang mengubah `self.__secret`
- Di luar class, coba akses `__secret` langsung (akan error), lalu akses via name mangling `_SecretKeeper__secret` (harus berhasil)

```python
class SecretKeeper:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    keeper = SecretKeeper("my_secret")
    print(keeper.reveal())  # my_secret
    # print(keeper.__secret)  # AttributeError
    print(keeper._SecretKeeper__secret)  # harus berhasil
```

**Expected Output**:
```
my_secret
my_secret
```
