## Materi

### Konsep Dasar

Python menyediakan dua cara cepat untuk membuat class yang terutama menyimpan data:

- **`@dataclass`** (Python 3.7+): decorator yang otomatis generates `__init__`, `__repr__`, `__eq__`, dan method lain berdasarkan class attributes.
- **`namedtuple`** (lebih lama): factory function untuk membuat tuple subclass dengan named fields (immutable).

Keduanya mengurangi boilerplate code untuk class yang hanya menyimpan data.

### Detail Teknis

**1. Dataclasses (`@dataclass`)**

```python
from dataclasses import dataclass
from typing import List

@dataclass
class Point:
    x: float
    y: float

p = Point(1.5, 2.5)
print(p)  # Point(x=1.5, y=2.5) - auto __repr__
print(p.x, p.y)  # 1.5 2.5
print(p == Point(1.5, 2.5))  # True - auto __eq__

# Default values
@dataclass
class Person:
    name: str
    age: int = 0  # default value
    city: str = "Unknown"

alice = Person("Alice", 25, "Jakarta")
bob = Person("Bob")  # age=0, city="Unknown"
```

**2. Dataclass Features**

```python
from dataclasses import dataclass, field, field(default_factory=list)

@dataclass(order=True)  # generates __lt__, __le__, __gt__, __ge__
class Employee:
    name: str
    age: int
    salary: float = field(default=0.0, compare=False)  # exclude dari sorting
    skills: list = field(default_factory=list)  # untuk mutable default

    # Post-processing after __init__
    def __post_init__(self):
        self.name = self.name.title()  # capitalize name

emp1 = Employee("alice", 25, 50000)
emp2 = Employee("Bob", 30, 60000)
print(emp1 < emp2)  # True (compare by name then age, salary excluded)

# Frozen (immutable)
@dataclass(frozen=True)
class ImmutablePoint:
    x: float
    y: float

pt = ImmutablePoint(1, 2)
# pt.x = 3  # FrozenInstanceError
```

**3. Named Tuples**

```python
from collections import namedtuple

# Define namedtuple class
Point = namedtuple('Point', ['x', 'y'])  # atau 'x y' sebagai string
p = Point(1, 2)
print(p)          # Point(x=1, y=2)
print(p.x, p.y)   # 1 2
print(p[0])       # 1 (also indexable like tuple)

# Unpacking
x, y = p
print(x, y)  # 1 2

# Convert to dict
d = p._asdict()  # {'x': 1, 'y': 2}

# Replace (returns new instance, immutable)
p2 = p._replace(x=10)  # Point(x=10, y=2)

# Default values (Python 3.7+)
Point = namedtuple('Point', ['x', 'y'], defaults=[0, 0])
p = Point()  # Point(x=0, y=0)
```

**4. Comparison: Dataclass vs NamedTuple**

| Feature | Dataclass | NamedTuple |
|---------|-----------|------------|
| Mutability | Mutable (unless `frozen=True`) | Immutable |
| Inheritance | Supports inheritance | Subclass of tuple, limited |
| Default values | Easy with `=` or `field(default_factory=...)` | `defaults` parameter |
| Type hints | Full support | Supported (but tuple under the hood) |
| Methods | Can define methods freely | Can define methods but limited |
| Performance | Slightly slower (class) | Faster (tuple-like) |
| Use case | General data containers, mutable | Lightweight immutable records, compatible with tuple APIs |

**5. When to Use Which**

- Use **dataclass** untuk:
  - Data yang perlu dimodifikasi setelah creation
  - Butuh fitur kompleks: post-init, inheritance, custom methods
  - Work dengan JSON/serialization (easy to dict)
  - Python 3.7+ required

- Use **namedtuple** untuk:
  - Data yang harus immutable (thread-safe, hashable)
  - Need tuple-like performance & compatibility (unpacking, indexing)
  - Simple struct tanpa banyak behavior
  - Python 2.7+ compatibility

### Soal 1: 2D Vector dengan Dataclass

Buat `Vector2D` dataclass dengan:
- Attributes: `x: float`, `y: float`
- Method `magnitude()`: return sqrt(x² + y²)
- Method `normalize()`: return new Vector2D yang sudah di-normalize (unit vector)
- Method `dot(other)`: return dot product
- `__add__`, `__sub__` untuk penjumlahan/pengurangan vektor

```python
from dataclasses import dataclass
import math

@dataclass
class Vector2D:
    x: float
    y: float
    
    def magnitude(self):
        return math.sqrt(self.x**2 + self.y**2)
    
    def normalize(self):
        mag = self.magnitude()
        if mag == 0:
            return Vector2D(0, 0)
        return Vector2D(self.x / mag, self.y / mag)
    
    def dot(self, other):
        return self.x * other.x + self.y * other.y
    
    def __add__(self, other):
        return Vector2D(self.x + other.x, self.y + other.y)
    
    def __sub__(self, other):
        return Vector2D(self.x - other.x, self.y - other.y)
    
    def __repr__(self):
        return f"Vector2D({self.x:.2f}, {self.y:.2f})"

def main():
    v1 = Vector2D(3, 4)
    v2 = Vector2D(1, 2)
    print(f"v1 = {v1}, magnitude = {v1.magnitude():.2f}")  # 5.00
    print(f"v1 + v2 = {v1 + v2}")
    print(f"v1 dot v2 = {v1.dot(v2)}")
    unit = v1.normalize()
    print(f"Unit vector: {unit}, magnitude: {unit.magnitude():.2f}")  # 1.00
```

### Soal 2: Employee Record dengan NamedTuple

Buat `Employee` namedtuple dengan fields:
- `name` (str)
- `employee_id` (str)
- `department` (str)
- `salary` (float)
- `skills` (list of str) - gunakan `default_factory=list` untuk mutable default (tapi namedtuple immutable, jadi hati-hati: default mutable di namedtuple bisa shared antar instance! Lebih baik gunakan dataclass untuk mutable fields.

```python
from collections import namedtuple
from typing import List

# TODO: Buat Employee namedtuple dengan fields dan defaults
# name: "", employee_id: "", department: "General", salary: 0.0, skills: []

def main():
    emp1 = Employee("Alice", "E001", "Engineering", 75000, ["Python", "Go"])
    emp2 = Employee("Bob", "E002", "HR")  # default salary, skills=[]
    
    print(emp1)
    print(emp2.skills)  # []
    
    # Convert ke dict
    d = emp1._asdict()
    print(d)
    
    # Ganti department (create new instance)
    emp1_updated = emp1._replace(department="Product")
    print(emp1_updated)
```

**Catatan**: Untuk mutable fields seperti list, namedtuple kurang aman karena default mutable akan di-share. Lebih baik gunakan dataclass:

```python
from dataclasses import dataclass, field

@dataclass
class EmployeeDC:
    name: str
    employee_id: str
    department: str = "General"
    salary: float = 0.0
    skills: List[str] = field(default_factory=list)  # safe mutable default
```

### Soal 3: Configuration Data Class

Buat `Config` dataclass yang membaca dari dictionary (misal dari JSON) dan memiliki validation:

```python
from dataclasses import dataclass, field
from typing import Optional

@dataclass
class Config:
    host: str = "localhost"
    port: int = 8080
    debug: bool = False
    database_url: Optional[str] = None
    max_connections: int = field(default=100, metadata={"min": 1, "max": 1000})
    
    def __post_init__(self):
        # Validation
        if not (1 <= self.port <= 65535):
            raise ValueError("Port must be 1-65535")
        if self.max_connections < 1:
            raise ValueError("max_connections must be >= 1")
    
    @classmethod
    def from_dict(cls, data: dict):
        """Create Config from dictionary (e.g., loaded from JSON)"""
        return cls(**data)

def main():
    # Default
    cfg1 = Config()
    print(cfg1)
    
    # Custom
    cfg2 = Config(host="0.0.0.0", port=3000, debug=True)
    print(cfg2)
    
    # From dict
    data = {"host": "example.com", "port": 9000, "max_connections": 50}
    cfg3 = Config.from_dict(data)
    print(cfg3)
    
    # Invalid port
    try:
        Config(port=99999)
    except ValueError as e:
        print(f"Error: {e}")
```
