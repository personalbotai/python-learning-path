## Materi

### Konsep Dasar

**Profiling** adalah measuring program's performance (time, memory) to identify bottlenecks.  
**Optimization** adalah improving code based on profiling results.

**Rule:**
1. Don't optimize prematurely
2. Measure before optimize (profile to find real bottlenecks)
3. Optimize the bottlenecks (80/20 rule: 80% time spent in 20% code)

### Detail Teknis

**1. Time measurement with `time`**

```python
import time

start = time.time()
# code to measure
result = slow_function()
elapsed = time.time() - start
print(f"Elapsed: {elapsed:.3f}s")

# More precise (monotonic)
start = time.perf_counter()
# code
elapsed = time.perf_counter() - start
```

**2. `timeit` module (micro-benchmarks)**

```python
import timeit

# Single statement
time = timeit.timeit('"-".join(str(n) for n in range(100))', number=10000)
print(f"Time: {time:.3f}s")

# With setup
time = timeit.timeit(
    'sum(range(1000))',
    setup='pass',
    number=1000
)

# From command line: python -m timeit "sum(range(1000))"

# Timeit in code
def test_function():
    return sum(range(1000))

time = timeit.timeit(test_function, number=1000)
```

**3. `cProfile` - Function-level profiling**

```python
import cProfile
import pstats

def slow_function():
    total = 0
    for i in range(1000000):
        total += i
    return total

# Run profiler
profiler = cProfile.Profile()
profiler.enable()
result = slow_function()
profiler.disable()

# Print stats to console
profiler.print_stats(sort='cumulative')  # sort by cumulative time

# Or save to file
profiler.dump_stats("profile_output.pstats")

# Analyze with pstats
stats = pstats.Stats("profile_output.pstats")
stats.sort_stats('cumulative')  # sort by cumulative time
stats.print_stats(10)  # top 10 functions

# Stats fields:
# ncalls: number of calls
# tottime: total time in function (excluding sub-functions)
# percall: tottime / ncalls
# cumtime: cumulative time (including sub-functions)
# percall: cumtime / ncalls (if primitive)
# filename:lineno(function): where defined
```

**4. Profiling with context manager**

```python
from contextlib import contextmanager
import cProfile

@contextmanager
def profiler():
    pr = cProfile.Profile()
    pr.enable()
    yield
    pr.disable()
    pr.print_stats(sort='cumulative')

with profiler():
    slow_function()
```

**7. Common Bottlenecks & Optimizations**

```python
# BAD: Quadratic algorithm O(n²)
def find_duplicates(lst):
    duplicates = []
    for i in range(len(lst)):
        for j in range(i+1, len(lst)):
            if lst[i] == lst[j]:
                duplicates.append(lst[i])
    return duplicates

# GOOD: O(n) with set
def find_duplicates_fast(lst):
    seen = set()
    duplicates = set()
    for item in lst:
        if item in seen:
            duplicates.add(item)
        else:
            seen.add(item)
    return list(duplicates)

# BAD: String concatenation in loop (O(n²))
s = ""
for i in range(10000):
    s += str(i)  # creates new string each time!

# GOOD: Join list
parts = []
for i in range(10000):
    parts.append(str(i))
s = "".join(parts)

# GOOD: List comprehension (often fastest)
s = "".join(str(i) for i in range(10000))

# BAD: Repeated global/module lookup in loop
import math
def compute():
    result = 0
    for i in range(1000000):
        result += math.sqrt(i)  # global lookup each iteration
    return result

# GOOD: Local variable
def compute_fast():
    result = 0
    sqrt = math.sqrt  # local lookup
    for i in range(1000000):
        result += sqrt(i)
    return result
```

**8. Using `functools.lru_cache` for memoization**

```python
from functools import lru_cache

@lru_cache(maxsize=None)
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Without cache: O(2^n)
# With cache: O(n)
```

**9. Vectorization with NumPy (conceptual)**

```python
# SLOW: pure Python loops
result = []
for i in range(1000000):
    result.append(i * 2)

# FAST: NumPy vectorization
import numpy as np
arr = np.arange(1000000)
result = arr * 2  # C-level loop, much faster
```

### Soal 1: Profile a Function

Buat fungsi `process_data` yang:
- Membuat list 1 juta angka acak
- Filter genap
- Kuadratkan
- Sum

Profile dengan `cProfile` dan identifikasi bottleneck.

```python
import random
import cProfile
import pstats
from io import StringIO

def process_data():
    # Generate 1M random numbers
    numbers = [random.randint(1, 1000) for _ in range(1000000)]
    
    # Filter even
    evens = []
    for n in numbers:
        if n % 2 == 0:
            evens.append(n)
    
    # Square
    squared = []
    for n in evens:
        squared.append(n ** 2)
    
    # Sum
    total = 0
    for n in squared:
        total += n
    
    return total

def profile_function():
    profiler = cProfile.Profile()
    profiler.enable()
    result = process_data()
    profiler.disable()
    
    # Print top 10 by cumulative time
    s = StringIO()
    stats = pstats.Stats(profiler, stream=s)
    stats.sort_stats('cumulative')
    stats.print_stats(10)
    print(s.getvalue())
    
    return result

if __name__ == "__main__":
    profile_function()
```

### Soal 2: Compare Implementations

Bandingkan performance dua implementasi `word_frequency`:

```python
import timeit
from collections import Counter
import re

def word_frequency_loop(text):
    words = re.findall(r'\b\w+\b', text.lower())
    freq = {}
    for word in words:
        freq[word] = freq.get(word, 0) + 1
    return freq

def word_frequency_counter(text):
    words = re.findall(r'\b\w+\b', text.lower())
    return Counter(words)

# Sample text (baca dari file atau string panjang)
sample_text = "Python is great. " * 10000

# Measure
time_loop = timeit.timeit(lambda: word_frequency_loop(sample_text), number=10)
time_counter = timeit.timeit(lambda: word_frequency_counter(sample_text), number=10)

print(f"Loop: {time_loop:.3f}s")
print(f"Counter: {time_counter:.3f}s")

```

### Soal 3: Decorator untuk Auto-Profiling

Buat decorator `@profile_func` yang otomatis profile function dan print stats:

```python
import cProfile
import pstats
from io import StringIO
from functools import wraps

def profile_func(sort_by='cumulative', top_n=10):
    """
    Decorator: profile function and print stats.
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            profiler = cProfile.Profile()
            profiler.enable()
            result = func(*args, **kwargs)
            profiler.disable()
            
            s = StringIO()
            stats = pstats.Stats(profiler, stream=s)
            stats.sort_stats(sort_by)
            stats.print_stats(top_n)
            print(f"\nProfile for {func.__name__}:")
            print(s.getvalue())
            
            return result
        return wrapper
    return decorator

@profile_func(sort_by='tottime', top_n=5)
def slow_algorithm():
    # Some slow code
    total = 0
    for i in range(100000):
        total += i ** 2
    return total

slow_algorithm()
```
