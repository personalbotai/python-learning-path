## Tujuan Pembelajaran
Configure logging module, use different log levels, format logs, and file handlers.

## Materi

### Konsep Dasar
**Logging** menggantikan `print()` untuk production. Memungkinkan level keparahan, output ke file/console, formatting, dan rotasi.

**Basic Usage**:
```python
import logging

logging.debug("Debug message")
logging.info("Info message")
logging.warning("Warning message")
logging.error("Error message")
logging.critical("Critical message")
```
Default level adalah `WARNING`, jadi `debug` dan `info` tidak muncul.

**Configuration**:
```python
logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    filename='app.log',
    filemode='a'
)
```

**Loggers**:
```python
logger = logging.getLogger(__name__)
logger.debug("Message")
```
Setiap module bisa punya logger sendiri.

### Log Levels
- `DEBUG`: detail info, biasanya untuk debugging
- `INFO`: konfirmasi things work as expected
- `WARNING`: something unexpected happened, but program continues
- `ERROR`: serious problem, some functionality failed
- `CRITICAL`: very serious error, program may stop

### Handlers
- `StreamHandler()` → console (sys.stdout/stderr)
- `FileHandler(filename)` → file
- `RotatingFileHandler(filename, maxBytes, backupCount)` → rotate file when too big
- `TimedRotatingFileHandler` → rotate based on time
- Banyak handlers sekaligus (log ke console dan file)

### Formatting
Format string menggunakan placeholders:
- `%(asctime)s`: waktu
- `%(name)s`: logger name
- `%(levelname)s`: level name
- `%(message)s`: pesan
- `%(filename)s`, `%(lineno)d`: lokasi

### Detail Teknis
- `basicConfig()` hanya efektif jikabelum ada handlers
- Logger hierarchy: `logging.getLogger('A.B')` inherit settings dari `'A'` kecuali override
- `logger.propagate = False` untuk stop propagasi ke parent logger
- `logging.disable(level)` untuk disable log di bawah level tertentu
- `try/except` lalu `logger.exception()` otomatis include traceback
- `NullHandler()` untuk menghindari "No handlers could be found" warning
- `logging.config.dictConfig()` untuk konfigurasi kompleks via dict/YAML

from logging.handlers import RotatingFileHandler

def latihan_1():

    logger.setLevel(logging.DEBUG)

    # File handler (all levels) dengan rotation
    # Console handler (WARNING+)
    pass

if __name__ == "__main__":
