## Materi

### Konsep Dasar

**Object-Oriented Programming (OOP)** adalah paradigma yang mengorganisir kode sebagai objects yang berisi data (attributes) dan behavior (methods). Python adalah multi-paradigm, tapi OOP sangat didukung.

**Class vs Object:**
- **Class**: blueprint/template (definisi)
- **Object**: instance dari class (real thing)

```python
# Class definition
class Dog:
    pass  # empty class

# Creating objects (instances)
my_dog = Dog()
your_dog = Dog()

print(type(my_dog))  # <class '__main__.Dog'>
print(isinstance(my_dog, Dog))  # True
```

### Detail Teknis

**1. `__init__()` - Constructor/Initializer**

```python
class Person:
    def __init__(self, name, age):
        """Initialize new Person object"""
        self.name = name  # instance attribute
        self.age = age    # instance attribute

# Membuat instance (otomatis panggil __init__)
alice = Person("Alice", 25)
bob = Person("Bob", 30)

print(alice.name)  # Alice
print(bob.age)     # 30
```

**`self` parameter:**
- `self` adalah reference ke object instance yang sedang dibuat/dimethod
- Harus menjadi parameter pertama di setiap instance method
- Python automatically passes instance sebagai first argument

```python
class Counter:
    def __init__(self):
        self.count = 0
    
    def increment(self):
        self.count += 1  # akses attribute via self

c = Counter()
c.increment()
print(c.count)  # 1
```

**2. Instance Attributes & Methods**

```python
class BankAccount:
    # Class attribute (shared by all instances)
    bank_name = "SuperBank"
    
    def __init__(self, account_number, owner, balance=0):
        # Instance attributes (unique per instance)
        self.account_number = account_number
        self.owner = owner
        self.balance = balance
    
    # Instance method
    def deposit(self, amount):
        """Add money to account"""
        if amount <= 0:
            raise ValueError("Amount must be positive")
        self.balance += amount
        return self.balance
    
    def withdraw(self, amount):
        """Remove money from account"""
        if amount <= 0:
            raise ValueError("Amount must be positive")
        if amount > self.balance:
            raise ValueError("Insufficient funds")
        self.balance -= amount
        return self.balance
    
    def get_balance(self):
        """Return current balance"""
        return self.balance
    
    def __str__(self):
        """Human-readable string (used by print())"""
        return f"Account({self.account_number}, {self.owner}, balance=${self.balance})"
    
    def __repr__(self):
        """Unambiguous string (for debugging)"""
        return f"BankAccount(account_number={self.account_number!r}, owner={self.owner!r}, balance={self.balance})"

# Usage
acc1 = BankAccount("12345", "Alice", 1000)
acc2 = BankAccount("67890", "Bob", 500)

print(acc1)  # __str__: Account(12345, Alice, balance=$1000)
print(repr(acc2))  # __repr__: BankAccount(...)

acc1.deposit(500)
acc1.withdraw(200)
print(acc1.get_balance())  # 1300
```

**3. Special Methods (Dunder Methods)**

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        return f"Vector({self.x}, {self.y})"
    
    def __repr__(self):
        return f"Vector(x={self.x}, y={self.y})"
    
    def __add__(self, other):
        """Overload + operator"""
        if isinstance(other, Vector):
            return Vector(self.x + other.x, self.y + other.y)
        return NotImplemented
    
    def __eq__(self, other):
        """Overload == operator"""
        if isinstance(other, Vector):
            return self.x == other.x and self.y == other.y
        return False
    
    def __len__(self):
        """Overload len() function"""
        return 2  # vector 2D
    
    def __getitem__(self, index):
        """Enable indexing: v[0], v[1]"""
        if index == 0:
            return self.x
        elif index == 1:
            return self.y
        else:
            raise IndexError("Vector index out of range")

v1 = Vector(1, 2)
v2 = Vector(3, 4)
v3 = v1 + v2  # calls __add__
print(v3)  # Vector(4, 6)
print(v1 == v2)  # False
print(len(v1))  # 2
print(v1[0], v1[1])  # 1 2
```

**4. Class Methods & Static Methods**

```python
class Person:
    population = 0  # class attribute
    
    def __init__(self, name):
        self.name = name
        Person.population += 1
    
    # Instance method (gets self)
    def greet(self):
        return f"Hello, I'm {self.name}"
    
    # Class method (gets cls, operates on class itself)
    @classmethod
    def get_population(cls):
        return f"Total population: {cls.population}"
    
    # Static method (no self or cls, just utility)
    @staticmethod
    def is_adult(age):
        return age >= 18

print(Person.get_population())  # "Total population: 0"
alice = Person("Alice")
print(alice.greet())  # "Hello, I'm Alice"
print(Person.get_population())  # "Total population: 1"
print(Person.is_adult(20))  # True
```

**5. Property Decorators (Getters/Setters)**

```python
class Circle:
    def __init__(self, radius):
        self._radius = radius  # convention: underscore = private
    
    @property
    def radius(self):
        """Getter for radius"""
        return self._radius
    
    @radius.setter
    def radius(self, value):
        """Setter for radius with validation"""
        if value <= 0:
            raise ValueError("Radius must be positive")
        self._radius = value
    
    @property
    def area(self):
        """Computed property (read-only)"""
        return 3.14159 * self._radius ** 2

c = Circle(5)
print(c.radius)   # 5 (calls getter)
c.radius = 10     # calls setter
print(c.area)     # 314.159 (computed)
# c.area = 100    # AttributeError: can't set attribute (read-only)
```

**6. Inheritance**

```python
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        raise NotImplementedError("Subclasses must implement")
    
    def __str__(self):
        return f"{self.name} the Animal"

class Dog(Animal):  # Dog inherits from Animal
    def speak(self):  # Override parent method
        return "Woof!"
    
    def fetch(self):  # Dog-specific method
        return f"{self.name} fetches the ball"

class Cat(Animal):
    def speak(self):
        return "Meow!"

dog = Dog("Buddy")
cat = Cat("Whiskers")

print(dog)        # Buddy the Animal
print(dog.speak())  # Woof!
print(dog.fetch())  # Buddy fetches the ball
print(cat.speak())  # Meow!

# isinstance checks
print(isinstance(dog, Animal))  # True (Dog is a subclass of Animal)
```

**7. `super()` - Calling Parent Methods**

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"I'm {self.name}, {self.age} years old"

class Employee(Person):
    def __init__(self, name, age, employee_id, department):
        super().__init__(name, age)  # Call parent __init__
        self.employee_id = employee_id
        self.department = department
    
    def introduce(self):
        # Extend parent method
        parent_intro = super().introduce()
        return f"{parent_intro}, Employee ID: {self.employee_id}, Dept: {self.department}"

emp = Employee("Alice", 25, "E123", "Engineering")
print(emp.introduce())
# "I'm Alice, 25 years old, Employee ID: E123, Dept: Engineering"
```

### Soal 1: Simple Bank Account

Buat class `BankAccount` dengan:
- Attributes: `account_number` (str), `owner` (str), `balance` (float, default 0)
- Methods:
  - `deposit(amount)` - tambah balance, validasi amount > 0
  - `withdraw(amount)` - kurangi balance, validasi amount > 0 dan tidak exceed balance
  - `transfer(target_account, amount)` - transfer ke account lain
  - `get_balance()` - return balance
  - `__str__()` - return formatted string
  - `__repr__()` - return unambiguous representation

```python
class BankAccount:
    # TODO: implement
    pass

def main():
    acc1 = BankAccount("12345", "Alice", 1000)
    acc2 = BankAccount("67890", "Bob", 500)
    
    print(acc1)  # BankAccount(12345, Alice, $1000.00)
    acc1.deposit(200)
    acc1.withdraw(100)
    print(acc1.get_balance())  # 1100
    
    acc1.transfer(acc2, 300)
    print(acc1.get_balance())  # 800
    print(acc2.get_balance())  # 800
```
