# Reading and Writing Text Files
**ID**: `M07-L01`

## Tujuan Pembelajaran
Open files with open(), use read/write modes, context managers, and encoding parameters.

## Materi

### Konsep Dasar
File I/O di Python menggunakan built-in function `open()`.

**Modes**:
- `'r'`: read (default)
- `'w'`: write (truncate, create jika tidak ada)
- `'a'`: append (create jika tidak ada)
- `'x'`: exclusive creation (error jika file exists)
- `'b'`: binary mode
- `'t'`: text mode (default)
- `'+'`: update (read and write)

Contoh: `'rb'` (read binary), `'w+'` (read/write truncate), `'a+'` (read/append)

**Encoding**: Parameter `encoding` (misal `'utf-8'`). Default platform-dependent.

### Reading Methods
- `file.read(size=-1)`: bisa seluruh file atau sejumlah byte/char
- `file.readline()`: satu baris (termasuk newline)
- `file.readlines()`: list semua baris
- Iterasi langsung: `for line in file:` (recommended untuk large files)

### Writing Methods
- `file.write(string)`: tulis string
- `file.writelines(iterable)`: tulis setiap string dari iterable (tidak tambah newline otomatis)
- `print(..., file=file)` juga bisa

### Context Manager
Gunakan `with open(...) as f:` untuk otomatis close file, meskipun ada exception.

### Detail Teknis
- Text mode: `read()` mengembalikan `str`, `readlines()` list of `str`
- Binary mode: `read()` mengembalikan `bytes`, `readlines()` list of `bytes`
- `newline` parameter mengontrol newline translation (text mode)
- `open()` mengembalikan `TextIOWrapper` (text) atau `BufferedReader/Writer` (binary)
- `file.closed` attribute untuk cek status
- `file.tell()` dan `file.seek(offset, whence)` untuk random access (binary lebih akurat)
- Hati-hati dengan encoding errors: gunakan `errors='ignore'` atau `'replace'` jika perlu
