## Tujuan Pembelajaran
Use lambda expressions, pass functions as arguments, and understand map/filter/reduce.

## Materi

### Konsep Dasar
**Lambda Functions**: Anonymous functions (tidak memiliki nama) dengan sintaks singkat.
```python
lambda arguments: expression
```
Lambda hanya bisa berupa single expression (bukan statements). Umum dipakai saat butuh fungsi sederhana sementara.

**Higher-Order Functions**: Fungsi yang menerima fungsi lain sebagai argumen, atau mengembalikan fungsi.

**map()**: Terapkan fungsi ke setiap elemen iterable.
```python
map(func, iterable)  # returns iterator
list(map(func, iterable))  # convert ke list
```

**filter()**: Filter elemen berdasarkan predikat (fungsi yang mengembalikan boolean).
```python
filter(pred, iterable)  # keeps elements where pred(element) True
list(filter(pred, iterable))
```

**reduce()**: (dari `functools`) Akumulasi elemen dengan fungsi biner.
```python
from functools import reduce
reduce(func, iterable, initializer=None)
```
`func` menerima dua argumen (akumulator, next_item) dan mengembalikan nilai baru.

### Detail Teknis
- Lambda sering dipakai sebagai argument untuk `sorted()`, `map()`, `filter()`, `reduce()`, `key` functions
- `map()` dan `filter()` lazy (returns iterator) → hemat memory
- `reduce()` tidak built-in, harus import dari `functools`
- `reduce()` tanpa initializer: pertama kali menggunakan dua elemen pertama
- `reduce()` dengan initializer: initializer sebagai akumulator awal
- Alternatif modern: list comprehensions sering lebih readable daripada map/filter
- Higher-order functions memungkinkan functional programming style
