## Materi

### Konsep Dasar

Python memiliki beberapa cara untuk memformat string. **f-strings** (formatted string literals) adalah cara modern (Python 3.6+) yang paling readable dan cepat.

**1. f-strings dasar**
```python
name = "Alice"
age = 25

# Variabel langsung di dalam {}
print(f"Nama: {name}, Usia: {age}")  # Nama: Alice, Usia: 25

# Expressions
print(f"Tahun depan: {age + 1}")      # Tahun depan: 26
print(f"2 x usia: {age * 2}")         # 2 x usia: 50

# Function calls
print(f"Nama uppercase: {name.upper()}")  # Nama uppercase: ALICE
```

**2. Formatting angka**
```python
# Integer
n = 12345
print(f"Biasa: {n}")                   # 12345
print(f"Dengan separator: {n:,}")      # 12,345 (thousand separator)
print(f"Hex: {n:#x}")                  # 0x3039
print(f"Binary: {n:b}")                # 11000000111001

# Float
pi = 3.14159265
print(f"Default: {pi}")                # 3.14159265
print(f"2 decimal: {pi:.2f}")          # 3.14
print(f"5 decimal: {pi:.5f}")          # 3.14159
print(f"Width 10, 2 dec: {pi:10.2f}") # "      3.14" (right aligned)
print(f"Width 10, 2 dec, left: {pi:<10.2f}")  # "3.14      "
print(f"Width 10, 2 dec, center: {pi:^10.2f}") # "   3.14   "

# Percentage
ratio = 0.875
print(f"Percentage: {ratio:.1%}")      # 87.5%
```

**3. Alignment & Width**
```python
# Alignment inside width
text = "Hello"
print(f"|{text:10}|")   # "|Hello     |" (right default)
print(f"|{text:<10}|")  # "|Hello     |" (left)
print(f"|{text:>10}|")  # "|     Hello|" (right)
print(f"|{text:^10}|")  # "|  Hello   |" (center)

# Fill character (selain space)
print(f"|{text:_^10}|") # "|__Hello___|"
print(f"|{text:0>10}|") # "|00000Hello|"

# Multiple columns
data = [
    ("Alice", 25, 85.5),
    ("Bob", 30, 92.0),
    ("Charlie", 35, 78.5)
]
print(f"{'Name':<10} {'Age':>3} {'Score':>6}")
for name, age, score in data:
    print(f"{name:<10} {age:>3} {score:>6.1f}")
```

**4. Date & Time formatting**
```python
from datetime import datetime

now = datetime(2024, 1, 15, 14, 30, 45)
print(f"Default: {now}")  # 2024-01-15 14:30:45

# Custom format
print(f"Date: {now:%Y-%m-%d}")          # 2024-01-15
print(f"Time: {now:%H:%M:%S}")          # 14:30:45
print(f"Full: {now:%Y-%m-%d %H:%M}")    # 2024-01-15 14:30
print(f"Indonesian: {now:%d %B %Y}")    # 15 Januari 2024
```

**5. Debugging dengan f-strings (Python 3.8+)**
```python
x = 10
y = 20
z = x + y

# = modifier prints expression and its value
print(f"{z=}")            # z=30
print(f"{x=} + {y=} = {z=}")  # x=10 + y=20 = z=30

# Useful for quick debugging
def calculate(a, b):
    result = a * b + a - b
    print(f"{a=}, {b=}, {result=}")
    return result
```

**6. Nested f-strings & complex expressions**
```python
# You can put any expression inside {}
numbers = [1, 2, 3, 4, 5]
print(f"Sum: {sum(numbers)}")         # Sum: 15
print(f"Max: {max(numbers)}")         # Max: 5
print(f"List squared: {[x**2 for x in numbers]}")

# Nested formatting (format the result)
value = 123.456
print(f"Formatted: {value:{'.2f' if value > 100 else '.3f'}}")
# Conditional formatting: 2 decimals if >100, else 3 decimals
```

**7. Escaping braces**
```python
# To include literal braces, double them
print(f"{{ This is in braces }}")  # { This is in braces }
print(f"{{value}} = {value}")      # {value} = 123.456
```

### Detail Teknis

**1. Format Specification Mini-Language**

Full syntax: `{value:format_spec}`

```
format_spec ::= [[fill]align][sign][#][0][width][grouping_option][.precision][type]
```

- `fill`: any character (except `{` or `}`) for padding
- `align`: `<` left, `>` right, `^` center, `=` padding after sign
- `sign`: `+` show sign for positive, `-` default, space for leading space
- `#`: alternate form (0x for hex, 0o for octal, trailing dot for float)
- `0`: zero padding (only with numeric types)
- `width`: minimum field width
- `grouping_option`: `,` (comma) or `_` (underscore) for thousands separator
- `.precision`: for float (decimal places), for string (max chars)
- `type`: `d` decimal int, `b` binary, `o` octal, `x` hex, `e` scientific, `f` fixed, `g` general, `%` percent, `s` string (default)

**Examples:**
```python
n = -42
print(f"{n:010d}")      # -000000042 (width 10, zero-padded)
print(f"{n:+d}")        # -42 (explicit sign)
print(f"{n: d}")        # "-42" (space for positive, but n negative)
print(f"{n:#x}")        # -0x2a (hex with 0x)

f = 1234.5678
print(f"{f:,.2f}")      # 1,234.57 (comma grouping, 2 decimals)
print(f"{f:_.4f}")      # 1_234.5678 (underscore grouping)
print(f"{f:e}")         # 1.234568e+03 (scientific)
print(f"{f:g}")         # 1234.57 (general format)
```

**2. str.format() vs f-strings**

```python
# str.format() - older but still valid
print("Nama: {}, Usia: {}".format(name, age))
print("Nama: {n}, Usia: {a}".format(n=name, a=age))
print("{0} + {0} = {1}".format(5, 10))  # positional

# f-string lebih simple dan cepat
print(f"Nama: {name}, Usia: {age}")

# str.format() bisa dari dict
data = {"name": "Alice", "age": 25}
print("Nama: {name}, Usia: {age}".format(**data))
```

**3. %-formatting (legacy, hindari untuk new code)**
```python
# Old style (C printf-like)
print("Nama: %s, Usia: %d" % (name, age))
print("Float: %.2f" % pi)
print("Hex: %x" % 255)  # ff

# Disadvantages: less readable, error-prone (tuple mismatch), no expression support
# Gunakan hanya untuk compatibility dengan very old code
```
