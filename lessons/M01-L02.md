## Materi

### Konsep Dasar

Python memiliki tipe data built-in yang sangat kaya. Variabel di Python tidak perlu dideklarasikan dengan tipe explicitly — tipe data otomatis ditentukan dari nilai yang diberikan.

**Deklarasi Variabel:**
```python
# Integer
umur = 25
suhu = -5

# Float (desimal)
tinggi = 1.75
phi = 3.14159

# String
nama = "Alice"
pesan = 'Halo, Dunia!'

# Boolean
is_active = True
is_admin = False

# None (null value)
result = None
```

**Operasi Dasar:**
```python
# Arithmetic
a = 10
b = 3
print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3 (floor division)
print(a % b)   # 1 (modulus)
print(a ** b)  # 1000 (exponentiation)

# String operations
text = "Python"
print(text * 3)      # PythonPythonPython
print(text + "!")    # Python!
print(len(text))     # 6
print(text[0])       # 'P'
print(text[-1])      # 'n'
print(text[1:4])     # 'yth'
```

**Type Casting:**
```python
# Konversi tipe
x = "123"
y = int(x)        # 123 (int)
z = float(x)      # 123.0 (float)
s = str(y)        # "123" (string)
b = bool(x)       # True (bool - string non-empty = True)

# hati-hati dengan konversi yang tidak valid
int("12.5")       # ValueError!
int("abc")        # ValueError!
```

### Detail Teknis

- **Mutable vs Immutable**:
  - **Immutable**: int, float, str, bool, tuple — tidak bisa diubah setelah dibuat
  - **Mutable**: list, dict, set — bisa diubah (ditambah/hapus elemen)
  
- **`id()` function**: mengembalikan memory address objek
  ```python
  x = 42
  y = 42
  print(id(x) == id(y))  # True (integer caching)
  
  a = "hello"
  b = "hello"
  print(id(a) == id(b))  # True (string interning)
  ```

- **Assignment tidak men-copy nilai**:
  ```python
  a = [1, 2, 3]  # list (mutable)
  b = a          # b refer ke objek yang sama
  b.append(4)
  print(a)       # [1, 2, 3, 4] — a ikut berubah!
  ```
