## Tujuan Pembelajaran
Define classes, create objects (instances), understand __init__ and self.

## Materi

### Konsep Dasar
**Class**: Blueprint untuk membuat objects. Mendefinisikan atribut (data) dan methods (fungsi).
**Object**: Instance dari class.

**Defining Class**:
```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, I'm {self.name}"
```

**__init__ Method**:
- Constructor: dipanggil saat object dibuat
- `self` adalah reference ke object instance yang baru dibuat
- Atribut.instance didefinisikan dengan `self.attr = value`

**Creating Objects**:
```python
p = Person("Alice", 25)  # p adalah instance
```

### Self Parameter
- `self` harus menjadi parameter pertama di setiap instance method
- Dipanggil otomatis saat method dipanggil: `p.greet()` → `Person.greet(p)`
- Nama `self` hanya konvensi, bisa apa saja (tapi gunakan `self` untuk readability)

### Atribut dan Methods
- **Instance attributes**: `self.attr` (unik per instance)
- **Class attributes**: `ClassName.attr` (shared semua instance)
- **Instance methods**: `def method(self, ...)`
- Bisa tambah atribut setelah creation: `p.new_attr = value` (tapi tidak disarankan)

### Detail Teknis
- Class juga merupakan objek (instance of `type`)
- `__dict__` menyimpan atribut instance
- `isinstance(obj, Class)` cek apakah obj adalah instance dari Class
- `issubclass(Sub, Super)` cek inheritance
- `__module__` dan `__qualname__` untuk introspection
- Class bisa tanpa `__init__` (default tidak melakukan apa-apa)

## Latihan

### Soal 1
Buat class `BankAccount` dengan:
- Atribut instance: `account_holder` (str), `balance` (float, default 0.0)
- Method: `deposit(amount)` (tambah balance), `withdraw(amount)` (kurangi balance, tapi jika amount > balance, print "Insufficient funds" dan tidak ubah balance), `get_balance()` (kembalikan balance)
- Test dengan membuat beberapa account dan lakukan transaksi.

```python
class BankAccount:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    acc1 = BankAccount("Alice", 100)
    acc2 = BankAccount("Bob")
    acc1.deposit(50)
    acc1.withdraw(30)
    acc2.deposit(200)
    acc2.withdraw(250)  # harus print error
    print(acc1.get_balance())  # 120
    print(acc2.get_balance())  # 200
```

**Expected Output**:
```
Insufficient funds
120
200
```

### Soal 2
Buat class `Rectangle` dengan:
- `__init__(self, width, height)`
- Method `area()` mengembalikan luas
- Method `perimeter()` mengembalikan keliling
- Method `is_square()` mengembalikan True jika width == height

```python
class Rectangle:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    r1 = Rectangle(4, 5)
    print(r1.area())       # 20
    print(r1.perimeter())  # 18
    print(r1.is_square())  # False
    r2 = Rectangle(3, 3)
    print(r2.is_square())  # True
```

**Expected Output**:
```
20
18
False
True
```
