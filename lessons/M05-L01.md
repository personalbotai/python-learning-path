## Materi

### Konsep Dasar

Fungsi adalah blok kode yang dapat dipakai ulang. Fungsi mengelompokkan logika yang sering digunakan, membuat code lebih organized, maintainable, dan DRY (Don't Repeat Yourself).

**1. Defining & Calling Functions**
```python
# Function definition
def greet(name):
    """Return greeting message for name"""
    return f"Hello, {name}!"

# Function call
message = greet("Alice")  # "Hello, Alice!"
print(message)
```

**2. Parameters & Arguments**
```python
# Positional arguments
def add(a, b):
    return a + b

result = add(5, 3)  # 8 (5 -> a, 3 -> b)

# Keyword arguments (urutan tidak penting)
result = add(b=3, a=5)  # 8

# Default arguments
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

greet("Alice")          # "Hello, Alice!"
greet("Bob", "Hi")      # "Hi, Bob!"
greet("Charlie", "Good morning", "extra")  # Error! Terlalu banyak argumen
```

**3. Return Values**
```python
# Return single value
def square(x):
    return x * x

# Return multiple values (tuple)
def divide_and_remainder(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder  # tuple

q, r = divide_and_remainder(10, 3)  # q=3, r=1

# Return None (implicit)
def do_nothing():
    pass  # empty function

result = do_nothing()  # None

# Early return
def is_positive(num):
    if num <= 0:
        return False
    # lebih banyak kode
    return True
```

### Detail Teknis

**4. Variable Scope**
```python
# Global scope
global_var = 10

def func():
    # Local scope
    local_var = 20
    print(global_var)   # OK: bisa akses global
    print(local_var)    # OK: lokal

func()
# print(local_var)     # NameError! local_var hanya di dalam func

# Modifikasi global variable inside function
count = 0
def increment():
    # count += 1  # UnboundLocalError! Python anggap count sebagai local
    global count   # deklarasikan explicit sebagai global
    count += 1

increment()  # count = 1
```

**5. Docstrings (documentation)**
```python
def calculate_circle_area(radius):
    """
    Calculate area of a circle given radius.

    Parameters:
        radius (float): radius of the circle in cm

    Returns:
        float: area of the circle (π * r²)

    Example:
        >>> calculate_circle_area(5)
        78.53981633974483
    """
    import math
    return math.pi * radius ** 2

# Access docstring
print(calculate_circle_area.__doc__)
help(calculate_circle_area)  # tampilkan help di terminal
```

**6. Type Hints (Python 3.5+)**
```python
def add_numbers(a: int, b: int) -> int:
    """Add two integers and return result"""
    return a + b

# Type hints tidak enforce, hanya dokumentasi
# Bisa dicek dengan mypy linter
result: int = add_numbers(5, 3)  # OK

def process(name: str, age: int, active: bool = True) -> str:
    return f"{name} ({age}) - Active: {active}"
```

**7. *args and **kwargs (variable arguments)**
```python
# *args: tuple of positional arguments
def sum_all(*args):
    return sum(args)

sum_all(1, 2, 3)        # 6
sum_all(1, 2, 3, 4, 5)  # 15

# **kwargs: dict of keyword arguments
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="Jakarta")
# name: Alice
# age: 25
# city: Jakarta

# Kombinasi regular params + *args + **kwargs
def func(a, b, *args, **kwargs):
    print(f"a={a}, b={b}")
    print(f"args: {args}")
    print(f"kwargs: {kwargs}")

func(1, 2, 3, 4, 5, x=10, y=20)
# a=1, b=2
# args: (3, 4, 5)
# kwargs: {'x': 10, 'y': 20}
```

**8. Lambda Functions (anonymous functions)**
```python
# Lambda: single expression function
square = lambda x: x ** 2
print(square(5))  # 25

# Gunakan dengan map, filter, sorted
numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, numbers))  # [1, 4, 9, 16]
evens = list(filter(lambda x: x % 2 == 0, numbers))  # [2, 4]
sorted_by_last = sorted(["Alice", "Bob", "Charlie"], key=lambda s: s[-1])

# Lambda dengan multiple args
add = lambda a, b: a + b
print(add(5, 3))  # 8

# Lambda dalam fungsi lain
def apply_operation(func, x, y):
    return func(x, y)

result = apply_operation(lambda a, b: a * b, 4, 5)  # 20
```

**9. Function as First-Class Objects**
```python
def greet(name):
    return f"Hello, {name}"

def farewell(name):
    return f"Goodbye, {name}"

# Functions bisa disimpan dalam variabel
funcs = [greet, farewell]
for func in funcs:
    print(func("Alice"))

# Functions sebagai arguments
def process_names(names, formatter):
    return [formatter(name) for name in names]

names = ["Alice", "Bob"]
formatted = process_names(names, greet)  # ["Hello, Alice!", "Hello, Bob!"]

# Functions sebagai return values
def get_greeting_formal(formal=True):
    if formal:
        return lambda name: f"Good day, Mr./Ms. {name}"
    else:
        return lambda name: f"Hey {name}!"

greeting = get_greeting_formal(False)
print(greeting("Alice"))  # "Hey Alice!"
```
