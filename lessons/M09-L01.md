## Materi

### Konsep Dasar

**Iterator** adalah object yang memproduksi data satu per satu saat diminta. Python menggunakan iterator protocol: object harus memiliki method `__iter__()` (return iterator itself) dan `__next__()` (return next value atau raise `StopIteration`).

**Generator** adalah cara mudah membuat iterator menggunakan `yield` keyword atau generator expressions. Generator produces values on-demand (lazy), sangat efisien untuk data besar atau infinite sequences.

### Detail Teknis

**1. Iterator Protocol**

```python
# Built-in iterators
numbers = [1, 2, 3]
it = iter(numbers)  # get iterator
print(next(it))  # 1
print(next(it))  # 2
print(next(it))  # 3
# print(next(it))  # StopIteration

# Manual iteration
it = iter([10, 20, 30])
while True:
    try:
        value = next(it)
        print(value)
    except StopIteration:
        break

# for loop actually uses iterator protocol
for value in [1, 2, 3]:
    print(value)  # implicitly calls iter() and next()
```

**2. Custom Iterator Class**

```python
class Countdown:
    """Iterator yang menghitung mundur dari n ke 1"""
    def __init__(self, start):
        self.current = start
    
    def __iter__(self):
        return self  # iterator mengembalikan dirinya sendiri
    
    def __next__(self):
        if self.current <= 0:
            raise StopIteration
        value = self.current
        self.current -= 1
        return value

# Usage
for num in Countdown(5):
    print(num)  # 5,4,3,2,1

# Bisa juga dikonversi ke list
list(Countdown(3))  # [3,2,1]
```

**3. Generator Functions (`yield`)**

```python
def countdown(n):
    """Generator: lebih sederhana dari custom iterator class"""
    while n > 0:
        yield n
        n -= 1

# Generator function mengembalikan generator object (iterator)
gen = countdown(5)
print(gen)  # <generator object countdown at 0x...>

for num in gen:
    print(num)  # 5,4,3,2,1

# Generator hanya di-evaluate saat diminta (lazy)
def infinite_seq():
    num = 0
    while True:
        yield num
        num += 1

# infinite_seq() tidak akan pernah selesai jika di-loop
# Tapi bisa di-slice dengan itertools.islice
```

**4. Generator Expressions**

```python
# List comprehension (eager, creates full list)
squares_list = [x**2 for x in range(10)]  # list

# Generator expression (lazy)
squares_gen = (x**2 for x in range(10))  # generator object

# Consume generator
for sq in squares_gen:
    print(sq)

# Convert to list jika perlu
squares_list = list(squares_gen)

# Nested generator expressions
matrix = ((i+j for j in range(3)) for i in range(3))
for row in matrix:
    print(list(row))
```

**5. `yield` vs `return`**

```python
def with_return(n):
    return n  # function ends, value returned

def with_yield(n):
    yield n  # function paused, value yielded
    # Bisa continue setelah yield
    yield n * 2

# Generator maintains state between yields
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib = fibonacci()
print(next(fib))  # 0
print(next(fib))  # 1
print(next(fib))  # 1
print(next(fib))  # 2
```

**6. `send()` dan `throw()` pada Generator**

```python
def generator_with_receive():
    """Generator yang bisa menerima nilai via send()"""
    received = yield "Ready"  # first yield, receive nilai pertama
    print(f"Received: {received}")
    received = yield f"Processed {received}"
    print(f"Received: {received}")

gen = generator_with_receive()
print(next(gen))          # "Ready" (prime generator)
print(gen.send("Hello")) # send "Hello", yield "Processed Hello"
# gen.throw(ValueError)  # raise exception di dalam generator
gen.close()              # close generator
```

**7. `itertools` Module**

```python
import itertools

# Infinite iterators
count = itertools.count(start=10, step=2)  # 10,12,14,...
cycle = itertools.cycle(['A', 'B', 'C'])   # A,B,C,A,B,C,...
repeat = itertools.repeat(42, times=3)    # 42,42,42

# Finite iterators
# accumulate: running totals (or other binary function)
nums = [1, 2, 3, 4]
list(itertools.accumulate(nums))  # [1,3,6,10]

# chain: concatenate multiple iterables
list(itertools.chain('ABC', [1,2,3]))  # ['A','B','C',1,2,3]

# compress: filter elements by selector
data = ['a', 'b', 'c', 'd']
selectors = [True, False, True, False]
list(itertools.compress(data, selectors))  # ['a','c']

# dropwhile/takewhile
list(itertools.dropwhile(lambda x: x<5, [1,3,5,7,9]))  # [5,7,9]
list(itertools.takewhile(lambda x: x<5, [1,3,5,7,9]))  # [1,3]

# permutations/combinations
list(itertools.permutations('AB', 2))  # [('A','B'),('B','A')]
list(itertools.combinations('ABC', 2)) # [('A','B'),('A','C'),('B','C')]

# product: Cartesian product
list(itertools.product('AB', [1,2]))  # [('A',1),('A',2),('B',1),('B',2)]

# groupby (requires sorted data)
data = sorted([('a',1),('a',2),('b',3),('b',4)])
for key, group in itertools.groupby(data, key=lambda x: x[0]):
    print(key, list(group))
# a [('a',1),('a',2)]
# b [('b',3),('b',4)]

# islice: slice iterator
list(itertools.islice(range(10), 2, 6, 2))  # [2,4]
```

**8. Memory Efficiency**

```python
# List comprehension (memory heavy)
def read_large_file_list(filename):
    with open(filename) as f:
        lines = [line.strip() for line in f]  # loads all into memory
    return lines

# Generator expression (memory efficient)
def read_large_file_gen(filename):
    with open(filename) as f:
        for line in f:
            yield line.strip()  # one line at a time

# Process large file without loading all into memory
def count_lines(filename):
    return sum(1 for _ in read_large_file_gen(filename))
```

**9. Coroutines (Advanced Generators)**

```python
def coroutine():
    """Generator yang bisa menerima dan memproses nilai"""
    while True:
        value = yield  # receive value
        print(f"Processing: {value}")

coro = coroutine()
next(coro)  # prime to first yield
coro.send(10)  # send 10
coro.send(20)  # send 20
coro.close()
```
