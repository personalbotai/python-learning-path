## Tujuan Pembelajaran

## Materi

### Konsep Dasar
**contextlib**: Module yang menyediakan utilities untuk bekerja dengan context managers.

**@contextmanager**: Decorator untuk membuat context manager dari generator function.
```python
from contextlib import contextmanager

@contextmanager
def my_context():
    # setup
    resource = acquire_resource()
    try:
        yield resource
    finally:
        # cleanup
        release_resource(resource)
```
Generator harus `yield` tepat sekali; cleanup code setelah `yield` di `finally` untuk ensure execution meskipun exception.

**contextlib utilities**:
- `contextlib.suppress(*exceptions)`: Men-suppress exceptions tertentu di dalam block.
  ```python
  with suppress(FileNotFoundError):
      os.remove('temp.txt')
  ```
- `contextlib.redirect_stdout(new_target)`: Redirect `sys.stdout` ke file-like object.
  ```python
  with open('output.txt', 'w') as f, redirect_stdout(f):
      print("Hello")  # goes to file
  ```
- `contextlib.redirect_stderr(new_target)`: Sama untuk stderr.
- `contextlib.closing(thing)`: Memastikan `thing.close()` dipanggil (alternative untuk `with thing:` jika thing tidak punya `__enter__/__exit__` tapi punya `close()`).
- `contextlib.ExitStack`: Mengelola context managers dinamis (tidak tahu jumlah sebelumnya).
  ```python
  with ExitStack() as stack:
      files = [stack.enter_context(open(f)) for f in filenames]
      # semua files akan ditutup saat exit
  ```

### Detail Teknis
- `@contextmanager` membuat class di belakang layar yang mengimplementasikan `__enter__` dan `__exit__`
- Exception yang terjadi di block diteruskan ke generator; jika tidak ditangkap di generator, akan propagate
- Di `__exit__` dari contextmanager, exception info passed sebagai args ke `try` block yang melingkari `yield`
- Bisa handle exception di generator: `try: yield ... except SomeError: handle`
- `suppress` mengabaikan exception yang cocok; exception lain propagate
- `redirect_stdout` dan `redirect_stderr` mengubah global `sys.stdout`/`sys.stderr` sementara
- `ExitStack` juga memiliki `pop_all()` untuk transfer context managers ke stack lain (advanced)

## Latihan

### Soal 1
Buat context manager `temp_file(content=None)` menggunakan `@contextmanager` yang:
- Membuat temporary file (gunakan `tempfile.NamedTemporaryFile(mode='w+', delete=False)`) agar file tidak auto-delete (kita ingin cek isi setelah block)
- Jika `content` diberikan, tulis ke file
- Yield nama file (path)
- Setelah block, file dihapus dengan `os.remove()`

```python
import os
import tempfile
from contextlib import contextmanager

@contextmanager
def latihan_1(content=None):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    with latihan_1("Hello World") as fname:
        with open(fname) as f:
            print(f.read())  # Hello World
    # setelah with, file sudah dihapus
    print(os.path.exists(fname))  # False
```

**Expected Output**:
```
Hello World
False
```

### Soal 2
Gunakan `contextlib.suppress` untuk menangani exception saat menghapus beberapa file yang mungkin tidak ada. Buat list file ['a.txt', 'b.txt', 'c.txt'] (hanya 'b.txt' yang ada). Gunakan `suppress(FileNotFoundError)` di dalam loop.

```python
from contextlib import suppress

def latihan_2():
    files = ['a.txt', 'b.txt', 'c.txt']
    # Buat file b.txt dulu untuk demo
    open('b.txt', 'w').close()
    # Tulis kode kamu di sini
    # Hapus semua file dengan suppress

if __name__ == "__main__":
    latihan_2()
    # Pastikan tidak ada error meskipun a.txt dan c.txt tidak ada
```

**Expected Output**: (no output, no errors)

### Soal 3
Gunakan `contextlib.ExitStack` untuk membuka multiple files (misal 3 file) dan memastikan semua ditutup. Buat list filenames, lalu `with ExitStack() as stack:` dan `files = [stack.enter_context(open(f)) for f in filenames]`. Cek bahwa semua file closed setelah exit.

```python
from contextlib import ExitStack

def latihan_3():
    filenames = ['file1.txt', 'file2.txt', 'file3.txt']
    # Buat file kosong dulu
    for f in filenames:
        open(f, 'w').close()
    # Tulis kode kamu di sini dengan ExitStack
    # Buka semua file, lalu di dalam block print status, setelah block cek closed

if __name__ == "__main__":
    latihan_3()
```

**Expected Output** (contoh):
```
Files opened.
All files closed? True
```
