# Instance Variables, Class Variables, and Methods
**ID**: `M05-L02`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Distinguish between instance and class attributes, define instance/class/static methods.

## Materi

### Konsep Dasar
**Instance Attributes**: `self.attr` - setiap instance memiliki nilai sendiri.
**Class Attributes**: `ClassName.attr` atau `self.__class__.attr` - dikelola bersama semua instance.

**Atribut precedence**: Jika instance memiliki attribute dengan nama yang sama dengan class attribute, instance attribute akan diutamakan.

**Types of Methods**:
- **Instance method**: `def method(self, ...)` - akses dan modifikasi instance & class attributes
- **Class method**: `@classmethod` def method(cls, ...) - akses class attributes, tidak bisa akses instance attributes (karena tidak ada `self`), biasanya untuk factory methods atau operasi tingkat class
- **Static method**: `@staticmethod` def method(...) - tidak menerima `self` atau `cls`, hanya fungsi biasa yang diletakkan di dalam class karena grouping

### Decorators
- `@classmethod` dan `@staticmethod` adalah decorators yang mengubah cara method di-call
- Class method menerima class sebagai argumen pertama (`cls`)
- Static method tidak menerima argumen khusus

### Detail Teknis
- Class attributes shared; perubahan melalui class akan terlihat di semua instance (kecuali instance override dengan own attribute)
- Class method bisa digunakan untuk membuat alternative constructors: `@classmethod def from_string(cls, s): ...`
- Static method untuk utility functions yang relevan dengan class tapi tidak perlu akses ke class/instance
- `cls` bisa digunakan untuk membuat instance baru: `obj = cls(...)`
- `__dict__` pada class berisi class attributes dan methods

## Latihan

### Soal 1
Buat class `Counter` yang:
- Memiliki class attribute `total_count` (int, awal 0) untuk menghitung total instance yang dibuat
- Setiap instance memiliki instance attribute `count` (awal 0)
- `__init__` meningkatkan `Counter.total_count` dan set `self.count = 0`
- Method `increment()` menaikkan `self.count` sebanyak 1
- Method `get_total_class_count()` (class method) mengembalikan `Counter.total_count`
- Method `get_instance_count()` (instance method) mengembalikan `self.count`

Test: Buat beberapa instance, panggil increment beberapa kali, cek total class count dan instance count.

```python
class Counter:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    c1 = Counter()
    c2 = Counter()
    c1.increment()
    c1.increment()
    c2.increment()
    print(c1.get_instance_count())  # 2
    print(c2.get_instance_count())  # 1
    print(Counter.get_total_class_count())  # 2
```

**Expected Output**:
```
2
1
2
```

### Soal 2
Buat class `MathUtils` dengan static methods:
- `is_even(n)` → True jika n genap
- `is_prime(n)` → True jika n bilangan prima (>=2)
- `factorial(n)` → factorial n (gunakan loop)

Jangan ada instance attributes, hanya static methods.

```python
class MathUtils:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    print(MathUtils.is_even(4))
    print(MathUtils.is_prime(7))
    print(MathUtils.factorial(5))
```

**Expected Output**:
```
True
True
120
```
