## Materi

### Konsep Dasar

List methods dapat dikategorikan menjadi:
- **Mutating methods**: mengubah list asli (in-place)
- **Non-mutating methods**: return list baru tanpa mengubah asli

**Methods penting:**

```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]

# Sorting
numbers.sort()                 # [1, 1, 2, 3, 4, 5, 6, 9] - in-place
sorted_numbers = sorted(numbers)  # return list baru

# Reverse
numbers.reverse()              # in-place
reversed_numbers = list(reversed(numbers))  # new list

# Count
count_1 = numbers.count(1)     # 2 (berapa kali 1 muncul)

# Index
idx = numbers.index(5)         # 4 (index pertama dari 5)
idx = numbers.index(5, 5)      # cari dari index 5 ke atas

# Remove duplicates (via set conversion)
unique = list(set(numbers))    # [1, 2, 3, 4, 5, 6, 9] - tapi urutan berubah!
```

### Detail Teknis

**1. Sorting dengan Custom Key**
```python
# Sort by panjang string
words = ["apple", "banana", "cherry", "date"]
words.sort(key=len)  # ['date', 'apple', 'banana', 'cherry']

# Sort by lowercase (case-insensitive)
words.sort(key=str.lower)  # ['apple', 'banana', 'cherry', 'date']

# Sort by custom function
def get_last_char(s):
    return s[-1]

words.sort(key=get_last_char)  # ['banana', 'apple', 'date', 'cherry']

# Lambda function (lebih ringkas)
words.sort(key=lambda s: s[-1])
```

**2. List sebagai Stack (LIFO)**
```python
stack = []
stack.append(1)    # [1]
stack.append(2)    # [1, 2]
stack.append(3)    # [1, 2, 3]
top = stack.pop()  # 3, stack: [1, 2]
```

**3. List sebagai Queue (FIFO) - menggunakan collections.deque**
```python
from collections import deque

queue = deque()
queue.append("pelanggan1")
queue.append("pelanggan2")
queue.append("pelanggan3")
first = queue.popleft()  # "pelanggan1"
```

**4. List slicing assignment (mengganti sebagian list)**
```python
nums = [0, 1, 2, 3, 4, 5]
nums[2:4] = [20, 30, 40]  # [0, 1, 20, 30, 40, 5] - bisa beda panjang
nums[1:3] = []             # [0, 40, 5] - hapus dengan empty list
```

**5. List concatenation & repetition**
```python
a = [1, 2, 3]
b = [4, 5, 6]

combined = a + b           # [1,2,3,4,5,6] - new list
a.extend(b)               # a modified: [1,2,3,4,5,6]

repeated = a * 3          # [1,2,3,1,2,3,1,2,3]
```

**6. Copying Lists**
```python
original = [1, 2, [3, 4]]

# Shallow copy - nested objects still shared!
shallow = original.copy()      # atau original[:] atau list(original)
shallow.append(5)              # [1,2,[3,4],5]
shallow[2].append(5)          # original[2] juga jadi [3,4,5]!

# Deep copy - semua level disalin
import copy
deep = copy.deepcopy(original)
deep[2].append(10)            # original tidak terpengaruh
```

**7. Unpacking Lists**
```python
first, middle, last = [1, 2, 3]  # unpack 3 elemen
print(first, middle, last)  # 1 2 3

# Starred expression
first, *rest, last = [1, 2, 3, 4, 5]
print(first)   # 1
print(rest)    # [2, 3, 4]
print(last)    # 5
```

**8. List Membership Test**
```python
items = ["apple", "banana", "cherry"]
print("banana" in items)     # True
print("grape" not in items)  # True

# Hati-hati: `in` pada list adalah O(n), pada set adalah O(1)
```
