## Tujuan Pembelajaran

## Materi

### Konsep Dasar
List comprehension adalah cara elegan untuk membuat list baru dari iterable dengan ekspresi.

**Syntax**:
```python
[expression for item in iterable]
[expression for item in iterable if condition]
[expression for item1 in iterable1 for item2 in iterable2]  # nested loops
```

**Equivalent with map/filter**:
- `[f(x) for x in iterable]` ≈ `list(map(f, iterable))`
- `[x for x in iterable if cond(x)]` ≈ `list(filter(cond, iterable))`

### Advanced Manipulation
- **sort()**: `list.sort(key=func, reverse=bool)` - in-place sorting
- **sorted()**: `sorted(list, key=func, reverse=bool)` - returns new sorted list
- **key function**: fungsi yang mengembalikan nilai untuk perbandingan
- **lambda**: sering dipakai untuk key function sederhana

### Detail Teknis
- List comprehension lebih readable dan sering lebih cepat daripada loop eksplisit
- Bisa menggunakan unpacking: `[x+y for x, y in pairs]`
- Conditional di akhir (filter) bisa dengan `if-else`: `[x if x>0 else 0 for x in nums]`
- Nested comprehensions untuk nested lists
- Map dan filter masih useful ketika bekerja dengan banyak data (lazy evaluation dengan itertools)
- `sorted()` bisa sorting tipe apapun asal bisa dibandingkan (dengan key function)
