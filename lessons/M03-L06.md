## Tujuan Pembelajaran
Create sets, perform set operations (union, intersection, difference), and understand set comprehensions.

## Materi

### Konsep Dasar
Set adalah unordered collection dari unique elements (tidak ada duplikat). Set mutable (bisa tambah/hapus), tapi elemennya harus hashable (immutable).

**Creating Sets**:
- `s = {1, 2, 3}` (literal)
- `s = set(iterable)` (constructor)
- Empty set: `set()` (bukan `{}` yang berarti dict)
- `frozenset` adalah immutable set

**Set Operations** (binary operators):
- Union: `s1 | s2` atau `s1.union(s2)`
- Intersection: `s1 & s2` atau `s1.intersection(s2)`
- Difference: `s1 - s2` (elements in s1 not in s2)
- Symmetric Difference: `s1 ^ s2` (elements in either but not both)

**Set Methods**:
- `add(elem)`: tambah satu elemen
- `update(iterable)`: tambah banyak elemen
- `remove(elem)`: hapus elemen, KeyError jika tidak ada
- `discard(elem)`: hapus elemen, tidak error jika tidak ada
- `pop()`: hapus dan kembalikan arbitrary elemen (KeyError jika kosong)
- `clear()`: kosongkan set
- `issubset(other)`, `issuperset(other)`, `isdisjoint(other)`

### Set Comprehensions
```python
{expr for item in iterable if condition}
```
Mirip list comprehension tapi dengan curly braces.

### Detail Teknis
- Set unordered: tidak ada index, tidak bisa di-access by index
- Element unik: duplikat otomatis dieliminasi saat create/update
- Membership test `in` sangat cepat (O(1) average) → berguna untuk deduplication dan membership check
- Set operations efisien: union O(len(s1)+len(s2)), intersection O(min(len(s1), len(s2)))
- `frozenset` bisa digunakan sebagai dictionary key (karena immutable)
- Set bisa berisi tipe混合 asalkan hashable
