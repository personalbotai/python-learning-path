Create lists, access elements, slicing, common methods (append, extend, insert, remove, pop).

## Materi

### Konsep Dasar
List adalah mutable sequence yang dapat berisi elemen dengan tipe berbeda.

**Creating Lists**:
- Literal: `my_list = [1, 2, 3]`
- Constructor: `my_list = list()` atau `list(iterable)`
- Empty list: `[]` atau `list()`

**Indexing & Slicing**:
- Indexing: `list[i]` (negatif: -1 terakhir)
- Slicing: `list[start:stop:step]` (stop exclusive)
- Out of range: IndexError

### Basic List Methods
- `append(x)`: tambah elemen di akhir
- `extend(iterable)`: tambah semua elemen dari iterable
- `insert(i, x)`: sisipkan di posisi i
- `remove(x)`: hapus elemen pertama dengan nilai x (ValueError jika tidak ada)
- `pop([i])`: hapus dan kembalikan elemen di i (default terakhir)
- `clear()`: hapus semua elemen
- `index(x)`: indeks pertama dengan nilai x
- `count(x)`: jumlah kemunculan x
- `sort(key=None, reverse=False)`: urutkan in-place
- `reverse()`: balik urutan in-place

### Detail Teknis
- List mutable: elemen bisa diubah, ditambah, dihapus
- Slicing menghasilkan list baru (shallow copy)
- `pop()` tanpa argumen menghapus elemen terakhir (LIFO)
- `remove()` mencari dari awal, hanya menghapus satu
- `sort()` menggunakan Timsort (stable, O(n log n) worst-case)
- `list.copy()` untuk shallow copy (sama dengan `list[:]`)
