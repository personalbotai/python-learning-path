# Defining and Calling Functions
**ID**: `M04-L01`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Write functions with def, understand parameters and return values, scope rules (LEGB).

## Materi

### Konsep Dasar
Function adalah blok kode yang dapat dipanggil berulang. Dengan `def` kita mendefinisikan function.

**Defining Functions**:
```python
def nama_fungsi(parameter1, parameter2, ...):
    """docstring optional"""
    # body
    return nilai  # optional, default None
```

**Calling Functions**:
- `nama_fungsi(arg1, arg2)`
- Positional arguments: urutan matters
- Keyword arguments: `nama_fungsi(param=value)` bisa acak urutan
- Bisa campur: positional dulu, baru keyword

**Return**:
- `return` mengembalikan nilai dan keluar dari fungsi
- Tanpa `return` atau `return None` mengembalikan `None`
- Bisa return multiple values (sebenarnya tuple): `return a, b, c`

### Scope: LEGB Rule
Python mencari nama dalam urutan:
1. **Local**: dalam fungsi saat ini
2. **Enclosing**: dalam fungsi luar (closure)
3. **Global**: di module level
4. **Built-in**: nama built-in Python (len, print, dll)

**global** dan **nonlocal** statements bisa digunakan untuk memodifikasi variabel di scope yang lebih luas.

### Detail Teknis
- Function adalah objek first-class: bisa disimpan ke variabel, passed sebagai argumen, dikembalikan
- Default arguments dievaluasi sekali saat definisi (hati-hati dengan mutable default)
- Docstring: `func.__doc__` menyimpan string dokumentasi
- Function annotations (PEP 484) untuk type hints: `def f(x: int) -> str:`
- Positional-only parameters (`/`) dan keyword-only (`*`) tersedia di Python 3.8+

- Jika tidak ada atau keduanya, kembalikan None
- Gunakan keyword arguments

```python
def latihan_1(celsius=None, fahrenheit=None):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    print(latihan_1(celsius=0))
    print(latihan_1(fahrenheit=212))
```

**Expected Output**:
```
32.0
100.0
```
