## Materi

### Konsep Dasar

String adalah sequence of characters. Di Python, string adalah **immutable** (tidak bisa diubah setelah dibuat). Ditulis dengan single quotes `'...'`, double quotes `"..."`, atau triple quotes `"""..."""` untuk multiline.

**Membuat string:**
```python
# Single quotes
s1 = 'Hello World'

# Double quotes
s2 = "Hello World"

# Triple quotes (multiline)
s3 = """Hello
World
Multi-line"""

# Empty string
empty = ""
```

**String adalah sequence:**
```python
text = "Python"

# Indexing (dimulai dari 0)
print(text[0])   # 'P'
print(text[-1])  # 'n' (last character)

# Slicing
print(text[0:3])   # 'Pyt'
print(text[:3])    # 'Pyt'
print(text[3:])    # 'hon'
print(text[::2])   # 'Pto' (every 2nd char)
print(text[::-1])  # 'nohtyP' (reverse)
```

### Detail Teknis

**1. String Concatenation & Repetition**
```python
first = "Hello"
last = "World"

# Concatenation dengan +
full = first + " " + last  # "Hello World"

# Repetition dengan *
repeat = "Ha" * 3  # "HaHaHa"

# NOTE: `+` dan `*` membuat string baru (immutable)
# Untuk banyak concatenation, gunakan join() atau f-string
```

**2. String Methods Penting**
```python
s = "  Hello, World!  "

# Strip (hapus whitespace)
s.strip()       # "Hello, World!"
s.lstrip()      # "Hello, World!  "
s.rstrip()      # "  Hello, World!"

# Case conversion
s.lower()       # "  hello, world!  "
s.upper()       # "  HELLO, WORLD!  "
s.title()       # "  Hello, World!  "
s.capitalize()  # "  Hello, world!  "

# Split & Join
words = s.split()           # ['Hello,', 'World!'] (by whitespace)
words = s.split(',')        # ['  Hello', ' World!  '] (by comma)
joined = "-".join(words)    # "  Hello-World!  "

# Replace
s.replace("Hello", "Hi")   # "  Hi, World!  "

# Find & Index
s.find("World")            # 8 (position, -1 jika tidak ada)
s.index("World")           # 8 (error jika tidak ada)
s.count("l")               # 3 (berapa kali 'l' muncul)

# Startswith & Endswith
s.startswith("Hello")      # False (ada whitespace)
s.strip().startswith("Hello")  # True
s.endswith("!")            # True

# Is methods (checking)
"123".isdigit()            # True
"abc".isalpha()            # True
"Hello123".isalnum()       # True
"  ".isspace()             # True
"Hello World".istitle()    # False (setiap kata capitalized)
```

**3. F-Strings (Formatted String Literals) - Python 3.6+**
```python
name = "Alice"
age = 25

# Basic interpolation
print(f"Nama: {name}, Usia: {age}")

# Expressions di dalam {}
print(f"Tahun depan usia: {age + 1}")
print(f"Usia dua kali: {age * 2}")

# Formatting numbers
pi = 3.14159265
print(f"Pi: {pi:.2f}")        # "Pi: 3.14"
print(f"Pi: {pi:10.3f}")      # "Pi:       3.142" (width 10)

# Alignment
print(f"{name:<10}")          # left-align, width 10
print(f"{name:>10}")          # right-align
print(f"{name:^10}")          # center

# Date formatting
from datetime import datetime
now = datetime.now()
print(f"Today is {now:%Y-%m-%d}")  # "Today is 2024-01-15"
```

**4. %-formatting (legacy, tidak disarankan)**
```python
# F-string lebih modern dan readable
name = "Bob"
age = 30
print("Nama: %s, Usia: %d" % (name, age))  # old style
print(f"Nama: {name}, Usia: {age}")        # new style (prefer)
```

**5. str.format() method (sebelum f-string)**
```python
print("Nama: {}, Usia: {}".format(name, age))
print("Nama: {n}, Usia: {a}".format(n=name, a=age))
print("Usia dua kali: {a}".format(a=age*2))
```

**6. Raw Strings (untuk regex, Windows paths)**
```python
# Normal string - backslash jadi escape
path = "C:\new\test.txt"  # \n dan \t di-interpret

# Raw string - backslash literal
path = r"C:\new\test.txt"  # "C:\new\test.txt"

# Useful untuk regex
import re
pattern = r"\d+\.\d+"  # match floating point
```

**7. Multiline Strings & Docstrings**
```python
# Triple quotes untuk string panjang
story = """Once upon a time,
there was a programmer
who learned Python."""

# Docstring (string literal pertama di module/function/class)
def my_function():
    """This is a docstring.
    Bisa multiple line.
    """
    pass

print(my_function.__doc__)  # access docstring
```

**8. String Immutability**
```python
s = "Hello"
# s[0] = "h"  # TypeError! Tidak bisa modify

# Tapi bisa reassign variabel
s = "World"  # OK - s now refers to new string object

# Untuk "modifikasi", create string baru
s = "Hello"
s = "h" + s[1:]  # "hello" - create new string
```
