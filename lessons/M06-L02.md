# Raising Exceptions and Custom Exceptions
**ID**: `M06-L02`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Raise exceptions with raise, define custom exception classes by inheriting from Exception.

## Materi

### Konsep Dasar
**Raising Exceptions**: Gunakan `raise` untuk memicu exception.
```python
raise ValueError("Invalid value")
raise MyCustomError("Something went wrong")
```

**Custom Exceptions**: Buat class sendiri yang inherit dari `Exception` (atau subclass lain).
```python
class MyError(Exception):
    def __init__(self, message, code=None):
        super().__init__(message)
        self.code = code
```

### Exception Chaining
Saat menangkap exception dan raise baru, gunakan `raise NewError() from original_exception` untuk preserve traceback.

```python
try:
    risky_operation()
except SomeError as e:
    raise MyError("Operation failed") from e
```

### When to Raise
- Validasi input: jika parameter tidak valid, `raise ValueError`
- Jika kondisi tidak terpenuhi, `raise RuntimeError` atau custom
- Untuk menandakan error yang tidak bisa ditangani di level saat ini
- Custom exception membantu differentiate error types

### Detail Teknis
- `raise` tanpa arguments di dalam except block akan re-raise exception yang sedang ditangkap
- Custom exception class bisa menambahkan attributes tambahan (seperti error code)
- `Exception` adalah base class untuk几乎所有 built-in exceptions yang bisa di-catch
- `BaseException` lebih tinggi (termasuk SystemExit, KeyboardInterrupt) - jarang di-raise manual
- `__str__` bisa di-override di custom exception untuk custom message
- Bisa raise instance atau class: `raise ValueError("msg")` atau `raise ValueError` (tapi dengan message lebih baik)

## Latihan

### Soal 1
Buat custom exception `InvalidAgeError` yang inherit dari `Exception`. Buat fungsi `set_age(age)` yang:
- Jika age < 0 atau age > 150, raise `InvalidAgeError("Age must be between 0 and 150")`
- Jika success, kembalikan age

```python
class InvalidAgeError(Exception):
    # Tulis kode kamu di sini
    pass

def latihan_1(age):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    try:
        print(latihan_1(25))
        print(latihan_1(-5))
    except InvalidAgeError as e:
        print(f"Caught: {e}")
```

**Expected Output**:
```
25
Caught: Age must be between 0 and 150
```

### Soal 2
Buat fungsi `process_data(data)` yang:
- Jika data adalah None, raise `ValueError("Data cannot be None")`
- Jika data bukan list atau tuple, raise `TypeError("Data must be list or tuple")`
- Jika list/tuple kosong, raise `RuntimeError("Empty data")`
- Jika success, kembalikan `len(data)`
- Di caller, tangkap semua exceptions dengan try-except dan print message-nya.

```python
def latihan_2(data):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    for test in [None, "string", [], [1,2,3]]:
        try:
            result = latihan_2(test)
            print(f"Result: {result}")
        except Exception as e:
            print(f"Error ({type(e).__name__}): {e}")
```

**Expected Output**:
```
Error (ValueError): Data cannot be None
Error (TypeError): Data must be list or tuple
Error (RuntimeError): Empty data
Result: 3
```
