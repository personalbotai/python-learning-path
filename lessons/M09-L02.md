## Materi

### Konsep Dasar

**Concurrency**: Eksekusi multiple tasks secara overlapping (bisa sequential atau parallel).  
**Parallelism**: Eksekusi multiple tasks secara simultaneous (multiple CPU cores).

Python memiliki **GIL (Global Interpreter Lock)** yang memastikan hanya satu thread yang mengeksekusi Python bytecode pada satu waktu. Ini berarti CPU-bound Python code tidak mendapat benefit dari multi-threading. Namun, threading tetap berguna untuk I/O-bound tasks (network, file I/O, database).

**Threading Use Cases:**
- I/O-bound operations (API calls, file operations)
- Background tasks (GUI responsiveness)
- Waiting on external resources

**Multiprocessing Use Cases:**
- CPU-bound tasks (computation, data processing)
- Bypass GIL untuk true parallelism

### Detail Teknis

**1. Basic Threading**

```python
import threading
import time

def worker(name, delay):
    print(f"{name} starting")
    time.sleep(delay)
    print(f"{name} finished after {delay}s")

# Create threads
t1 = threading.Thread(target=worker, args=("Thread-1", 2))
t2 = threading.Thread(target=worker, args=("Thread-2", 1))

# Start threads
t1.start()
t2.start()

# Wait for completion
t1.join()
t2.join()
print("All threads done")
```

**2. Thread with Arguments & kwargs**

```python
def task(id, data=None):

thread = threading.Thread(target=task, args=(1,), kwargs={"data": "hello"})

```

**3. Daemon Threads (background)**

```python
def background_task():
    while True:
        time.sleep(1)
        print("Background...")

# Daemon thread: otomatis terminate saat main thread exit
t = threading.Thread(target=background_task, daemon=True)
t.start()
time.sleep(3)
print("Main exiting")  # daemon thread akan dimatikan otomatis
```

**4. Thread Synchronization - Lock**

```python
import threading

counter = 0
lock = threading.Lock()

def increment():
    global counter
    for _ in range(100000):
        with lock:  # acquire & release otomatis
            counter += 1

threads = [threading.Thread(target=increment) for _ in range(5)]
for t in threads:
    t.start()
for t in threads:
    t.join()

print(f"Counter: {counter}")  # 500000 (thread-safe)
```

**5. RLock (Reentrant Lock)** - lock yang bisa di-acquire multiple times oleh same thread

```python
rlock = threading.RLock()

def nested():
    with rlock:
        with rlock:  # OK: RLock bisa di-acquire ulang oleh thread yang sama
            print("Nested lock")
```

**6. Semaphore** - batasi concurrent access ke resource

```python
semaphore = threading.Semaphore(value=3)  # max 3 threads sekaligus

def limited_task():
    with semaphore:
        print(f"Working: {threading.current_thread().name}")
        time.sleep(1)

threads = [threading.Thread(target=limited_task) for _ in range(10)]
for t in threads:
    t.start()
for t in threads:
    t.join()
# Maksimal 3 threads akan print secara bersamaan
```

**7. Event** - communication between threads

```python
event = threading.Event()

def waiter():
    print("Waiting for event...")
    event.wait()  # block sampai event.set()
    print("Event received!")

def setter():
    time.sleep(2)
    print("Setting event")
    event.set()

t1 = threading.Thread(target=waiter)
t2 = threading.Thread(target=setter)
t1.start()
t2.start()
t1.join()
t2.join()
```

**8. Condition** - complex coordination

```python
condition = threading.Condition()
queue = []

def producer():
    with condition:
        queue.append("item")
        condition.notify()  # wake up consumer

def consumer():
    with condition:
        while not queue:
            condition.wait()  # release lock & sleep until notified
        item = queue.pop()
        print(f"Consumed: {item}")
```

**9. concurrent.futures.ThreadPoolExecutor (high-level API)**

```python
from concurrent.futures import ThreadPoolExecutor, as_completed

def download(url):
    time.sleep(1)  # simulate I/O
    return f"Content from {url}"

urls = ["url1", "url2", "url3", "url4", "url5"]

with ThreadPoolExecutor(max_workers=3) as executor:
    # Submit tasks
    future_to_url = {executor.submit(download, url): url for url in urls}
    
    # Process completed tasks
    for future in as_completed(future_to_url):
        url = future_to_url[future]
        try:
            result = future.result()  # get result (blocks if not done)
            print(f"{url} -> {result}")
        except Exception as e:
            print(f"{url} generated exception: {e}")

# Map function
with ThreadPoolExecutor() as executor:
    results = executor.map(download, urls)  # returns iterator
    for result in results:
        print(result)
```

**10. Thread Local Data**

```python
thread_local = threading.local()

def show():
    print(threading.current_thread().name, thread_local.value)

def worker(value):
    thread_local.value = value
    show()

threads = [
    threading.Thread(target=worker, args=("Alice",), name="Thread-A"),
    threading.Thread(target=worker, args=("Bob",), name="Thread-B")
]
for t in threads:
    t.start()
for t in threads:
    t.join()
# Thread-A Alice
# Thread-B Bob
```

**11. GIL Impact**

```python
# CPU-bound task (GIL prevents true parallelism)
def cpu_task(n):
    result = 0
    for i in range(n):
        result += i
    return result

# Single-threaded
start = time.time()
cpu_task(10**7)
print(f"Single: {time.time()-start:.2f}s")

# Multi-threaded (tidak lebih cepat karena GIL)
threads = [threading.Thread(target=cpu_task, args=(10**7,)) for _ in range(4)]
start = time.time()
for t in threads:
    t.start()
for t in threads:
    t.join()
print(f"Multi-thread: {time.time()-start:.2f}s")  # ~same or slower
```

**12. Multiprocessing (bypass GIL)**

```python
from multiprocessing import Pool

def cpu_task(n):
    result = 0
    for i in range(n):
        result += i
    return result

# Process pool (true parallelism)
with Pool(4) as pool:
    results = pool.map(cpu_task, [10**7]*4)
print(results)
```
