# Multiprocessing: True Parallel Execution
**ID**: `M09-L02`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Spawn processes with multiprocessing, share data with Queue and Pipe, bypass GIL for CPU-bound tasks.

## Materi

### Konsep Dasar
**Multiprocessing**: Membuat multiple processes, masing-masing memiliki memory space sendiri. Bypass GIL, memungkinkan true parallel execution untuk CPU-bound tasks.

**Creating Processes**:
```python
from multiprocessing import Process

def worker(name):
    print(f"Hello from {name}")

p = Process(target=worker, args=("Alice",))
p.start()
p.join()
```

**Process Pool**: `multiprocessing.Pool` untuk manage pool of worker processes.
```python
from multiprocessing import Pool

def square(x):
    return x * x

with Pool(4) as pool:
    results = pool.map(square, [1,2,3,4,5])
```

### Inter-Process Communication (IPC)
- **Queue**: `multiprocessing.Queue` untuk thread-safe dan process-safe queue.
- **Pipe**: `multiprocessing.Pipe()` untuk duplex communication (two-way).
- **Manager**: `multiprocessing.Manager()` untuk shared objects (list, dict, Namespace) di antara processes.
- **Value/Array**: Shared memory untuk primitive types.

### GIL Bypass
Karena setiap process memiliki interpreter GIL-nya sendiri, multiprocessing dapat memanfaatkan multiple cores untuk CPU-bound tasks (seperti komputasi, image processing). Namun overhead process creation dan serialization (pickle) lebih tinggi daripada threading.

### Detail Teknis
- Process identity: `p.pid` (process ID)
- `p.is_alive()`, `p.exitcode`
- `p.terminate()` untuk kill process
- `Pool.map()` blocks until all done; `pool.apply_async()` untuk async
- `multiprocessing.cpu_count()` untuk jumlah cores
- Data yang dikirim via Queue/Pipe harus pickleable
- Windows: `if __name__ == '__main__':` wajib untuk multiprocessing (tidak di Unix yang menggunakan fork)
- Shared memory: `multiprocessing.Value`, `Array` untuk efficient sharing
- `multiprocessing.Event`, `Lock`, `Semaphore` juga tersedia untuk synchronization antar processes

### Soal 2
Buat dua processes yang berkomunikasi menggunakan `Pipe`. Process A mengirim bilangan genap dari 0 hingga 10 (inclusive) ke Process B. Process B menerima dan mencetak setiap bilangan yang diterima.

```python
from multiprocessing import Process, Pipe

def sender(conn):
    # Tulis kode kamu di sini
    pass

def receiver(conn):
    # Tulis kode kamu di sini
    pass

def latihan_2():
    parent_conn, child_conn = Pipe()
    p1 = Process(target=sender, args=(parent_conn,))
    p2 = Process(target=receiver, args=(child_conn,))
    p1.start()
    p2.start()
    p1.join()
    p2.join()

if __name__ == "__main__":
    latihan_2()
```

**Expected Output** (order may vary):
```
Received: 0
Received: 2
Received: 4
Received: 6
Received: 8
Received: 10
```
