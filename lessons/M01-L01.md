## Materi

### Konsep Dasar

Python adalah bahasa pemrograman yang mudah dipelajari dan sangat ekspresif. Pertama-tama, kita perlu memastikan Python terinstal dengan benar.

**Cek instalasi Python:**
```bash
python3 --version
# Output: Python 3.x.x
```

**Jalakan Python interaktif (REPL):**
```bash
python3
>>> print("Hello, World!")
Hello, World!
>>> exit()
```

**Struktur script Python:**
```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

# Import library (jika diperlukan)
import sys

def main():
    """Fungsi utama program"""
    print("Hello, World!")
    print(f"Python version: {sys.version}")

if __name__ == "__main__":
    main()
```

### Detail Teknis

- **Shebang line** (`#!/usr/bin/env python3`): mengarahkan sistem untuk menggunakan Python interpreter
- **Encoding declaration**: opsional tapi berguna untuk kompatibilitas UTF-8
- **Fungsi `main()`**: best practice untuk menempatkan logika utama dalam fungsi
- **`if __name__ == "__main__"`**: mencegah kode berjalan saat di-import sebagai modul
- **f-string** (`f"..."`): cara modern formatting string di Python 3.6+

```python
    # Tampilkan platform
    # Tampilkan current working directory
```

**Expected Output**:
```
=== Environment Info ===
Python version: 3.x.x
Platform: linux (atau darwin, win32, dll)
Current directory: /path/to/current/dir
```
