## Tujuan Pembelajaran
Build dictionaries with comprehensions, use get(), setdefault(), update(), and understand defaultdict.

## Materi

### Konsep Dasar
**Dictionary Comprehension**: Mirip list comprehension tapi menghasilkan dictionary.

Syntax:
```python
{key_expr: value_expr for item in iterable}
{key_expr: value_expr for item in iterable if condition}
```

**Common Dict Methods**:
- `get(key, default=None)`: safe access tanpa KeyError
- `setdefault(key, default)`: get value, atau set default jika tidak ada
- `update(other)`: merge dictionary lain (bisa dict, tuple pairs, kwargs)
- `defaultdict(default_factory)`: dari `collections`, otomatis buat value default untuk key yang tidak ada

### defaultdict
- `from collections import defaultdict`
- `dd = defaultdict(int)` → default value 0
- `dd = defaultdict(list)` → default value empty list []
- `dd = defaultdict(str)` → default value ""
- `dd[key]` akan otomatis membuat entry dengan default jika key belum ada

### Detail Teknis
- Dictionary comprehension lebih readable dan sering lebih cepat daripada loop + assignment
- `setdefault` sering dipakai untuk grouping: `d.setdefault(key, []).append(value)`
- `update()` bisa menerima multiple dictionaries, keyword arguments, atau iterable of pairs
- `defaultdict` sangat berguna untuk grouping, counting (dengan `int`), dan nested dicts
- Hati-hati: `dd[key]` akan selalu membuat entry, bahkan saat cek `if key in dd` → `dd[key]` akan membuat default
