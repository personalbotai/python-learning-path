## Materi

### Konsep Dasar

Set adalah unordered collection dari **unique** (tidak ada duplikat) dan **hashable** elemen. Ditulis dengan `{...}` atau `set()`.

**Membuat set:**
```python
# Set kosong (hati-hati: {} adalah dictionary, bukan set!)
empty_set = set()

# Set dengan elemen
fruits = {"apel", "pisang", "jeruk", "apel"}  # duplikat otomatis dihapus
print(fruits)  # {'pisang', 'jeruk', 'apel'} - urutan tidak dijamin

# Set dari list/tuple/string
unique_numbers = set([1, 2, 2, 3, 4, 4])  # {1, 2, 3, 4}
unique_chars = set("hello")  # {'h', 'e', 'l', 'o'}
```

**Akses elemen:**
```python
# Set tidak support indexing (unordered)
fruits = {"apel", "pisang", "jeruk"}
# fruits[0]  # TypeError!

# Tapi bisa di-iterate
for fruit in fruits:
    print(fruit)
```

### Detail Teknis

**1. Set Methods Utama**
```python
s = {1, 2, 3, 4, 5}

# Add (tambah elemen)
s.add(6)          # {1,2,3,4,5,6}
s.add(3)          # no effect, 3 already exists

# Remove (hapus, error jika tidak ada)
s.remove(1)       # {2,3,4,5,6}
# s.remove(10)    # KeyError!

# Discard (hapus, tidak error jika tidak ada)
s.discard(10)     # aman, no error

# Pop (hapus & return arbitrary elemen)
elem = s.pop()    # hapus satu elemen (acak), return valuenya

# Clear
s.clear()         # set()

# Membership test (O(1) vs O(n) di list!)
print(3 in s)     # True (fast)
print(10 not in s) # True
```

**2. Set Operations Matematis**
```python
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

# Union (gabungan) - |
union = a | b    # {1,2,3,4,5,6,7,8}
union = a.union(b)

# Intersection (irisan) - &
intersection = a & b  # {4, 5}
intersection = a.intersection(b)

# Difference (elemen di a tapi tidak di b) - -
difference = a - b    # {1, 2, 3}
difference = a.difference(b)

# Symmetric Difference (elemen di salah satu, bukan keduanya) - ^
sym_diff = a ^ b      # {1,2,3,6,7,8}
sym_diff = a.symmetric_difference(b)

# Subset & Superset
c = {1, 2}
print(c.issubset(a))     # True (c ⊆ a)
print(a.issuperset(c))   # True (a ⊇ c)
print(a > c)             # True (proper superset)
print(a >= c)            # True (superset)
```

**3. Set Comprehensions**
```python
# Set dari range
squares = {x**2 for x in range(10)}  # {0,1,4,9,16,25,36,49,64,81}

# Dengan kondisi
even_squares = {x**2 for x in range(10) if x % 2 == 0}
# {0, 4, 16, 36, 64}

# Dari string
unique_chars = {c for c in "hello world" if c != ' '}
# {'d', 'e', 'h', 'l', 'o', 'r', 'w'}
```

**4. Frozen Set (immutable set)**
```python
# Set normal mutable
s = {1, 2, 3}
s.add(4)  # OK

# Frozen set immutable
fs = frozenset([1, 2, 3])
# fs.add(4)  # AttributeError! Tidak bisa diubah

# Tapi bisa jadi dictionary key (karena hashable)
d = {fs: "frozen set as key"}

# Frozen set methods (hanya read-only)
print(fs.intersection({2, 3, 4}))  # frozenset({2, 3})
```

**5. Removing Duplicates from List**
```python
# Cara cepat hapus duplikat + maintain uniqueness
duplicate_list = [1, 2, 2, 3, 4, 4, 5]
unique_list = list(set(duplicate_list))  # [1,2,3,4,5]
# Tapi urutan tidak dijamin!

# Jika perlu urutan original:
seen = set()
unique_ordered = []
for item in duplicate_list:
    if item not in seen:
        seen.add(item)
        unique_ordered.append(item)
```

**6. Finding Unique Elements Across Multiple Collections**
```python
list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]
list3 = [4, 5, 6, 7]

# Elemen yang muncul di semua list
common = set(list1) & set(list2) & set(list3)  # {4}

# Elemen yang muncul di minimal 2 list
from collections import Counter
all_items = list1 + list2 + list3
counts = Counter(all_items)
at_least_2 = {item for item, cnt in counts.items() if cnt >= 2}
# {3, 4, 5, 6}
```
