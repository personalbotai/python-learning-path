# Logical Operators and Boolean Expressions
**ID**: `M02-L03`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Combine conditions with and, or, not; understand truthiness and operator precedence.

## Materi

### Konsep Dasar
Logical operators digunakan untuk menggabungkan atau membalikkan kondisi boolean:

- `and`: True jika kedua operand True
- `or`: True jika salah satu operand True
- `not`: membalikkan nilai boolean

**Truthiness**: Nilai-nilai berikut dianggap False dalam konteks boolean:
- `False`, `None`
- Angka nol: `0`, `0.0`
- Sequence kosong: `""`, `""`, `[]`, `()`, `{}`, `set()`, `range(0)`
- Objek yang mengimplementasikan `__bool__()` yang mengembalikan False

Semua nilai lain dianggap True.

### Operator Precedence
Prioritas operator (dari tertinggi ke terendah):
1. `not`
2. `and`
3. `or`

Gunakan parentheses `()` untuk kontrol eksplisit dan readability.

### Detail Teknis
- Short-circuit evaluation: `and` berhenti jika pertama False; `or` berhenti jika pertama True
- Comparison operators (`==`, `!=`, `<`, `>`, dll) memiliki precedence lebih tinggi daripada `not`
- `is` dan `in` juga comparison operators
- `bool()` function untuk konversi eksplisit ke boolean
- Truthiness umumnya mengikuti `__bool__()` atau `__len__()` (0 = False)
