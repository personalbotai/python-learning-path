## Materi

### Konsep Dasar

Tuple adalah sequence ordered dan **immutable** (tidak bisa diubah setelah dibuat). Ditulis dengan `(...)` atau tanpa parentheses dalam beberapa konteks.

**Membuat tuple:**
```python
# Tuple kosong
empty_tuple = ()

# Tuple dengan elemen
coordinates = (10, 20)
single_element = (42,)      # trailing comma WAJIB untuk single element
single_element = 42,        # juga valid (tanpa parentheses)
person = ("Alice", 25, "Jakarta")

# Tuple dari list/tuple lain
t = tuple([1, 2, 3])  # (1, 2, 3)
```

**Akses elemen:**
```python
person = ("Alice", 25, "Jakarta")
print(person[0])   # "Alice"
print(person[-1])  # "Jakarta"
print(person[1:])  # (25, "Jakarta") - slicing menghasilkan tuple baru
```

**Immutability:**
```python
t = (1, 2, 3)
# t[0] = 10      # TypeError! Tidak bisa assignment
# t.append(4)    # AttributeError! Tidak ada append
# t.sort()       # AttributeError! Tidak ada sort
```

### Detail Teknis

**1. Tuple Unpacking**
```python
# Basic unpacking
x, y, z = (1, 2, 3)
print(x, y, z)  # 1 2 3

# Swap tanpa temporary variable
a = 10
b = 20
a, b = b, a  # a=20, b=10

# Unpacking dengan star (*)
first, *middle, last = [1, 2, 3, 4, 5]
print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5

# Ignore values dengan underscore
_, year, month = (2024, 2023, 12)  # hanya butuh year dan month
```

**2. Tuple vs List - Kapan Pakai?**
```python
# Gunakan LIST jika:
# - Data perlu diubah (tambah/hapus/ubah elemen)
# - Bersifat koleksi yang seragam (semua elemen jenis sama)
# - Iterasi yang sering modifikasi

# Gunakan TUPLE jika:
# - Data harus tetap konstan (immutable) - protection
# - Penggunaan sebagai dictionary keys (hashable)
# - Multiple return values dari fungsi
# - Struct/record ringan
# - Performa: tuple sedikit lebih cepat dan hemat memory
```

**3. Tuple sebagai Multiple Return Values**
```python
def split_name(full_name):
    parts = full_name.split()
    return parts[0], parts[-1]  # return tuple

first, last = split_name("Alice Smith")
print(first)  # Alice
print(last)   # Smith

# Atau
result = split_name("Bob Johnson")
print(result)  # ('Bob', 'Johnson')
```

**4. Named Tuples (struct ringan)**
```python
from collections import namedtuple

# Definisi
Point = namedtuple('Point', ['x', 'y'])
# atau Point = namedtuple('Point', 'x y')

# Penggunaan
p1 = Point(10, 20)
print(p1.x, p1.y)        # 10 20
print(p1[0], p1[1])      # 10 20 (juga bisa di-index)
x, y = p1                # unpacking

# Convert ke dictionary
print(p1._asdict())      # {'x': 10, 'y': 20}

# Replace (return new instance)
p2 = p1._replace(x=30)  # Point(x=30, y=20)
```

**5. Tuple Methods (sedikit karena immutable)**
```python
t = (1, 2, 3, 2, 1)

t.count(1)      # 2 (berapa kali 1 muncul)
t.index(2)      # 1 (index pertama 2)
# t.index(10)   # ValueError jika tidak ada
```

**6. Tuple Concatenation & Repetition**
```python
a = (1, 2, 3)
b = (4, 5)

c = a + b       # (1, 2, 3, 4, 5) - new tuple
d = a * 3       # (1,2,3,1,2,3,1,2,3)
```

**7. Tuples dalam Dictionary Keys**
```python
# List tidak bisa jadi key karena mutable (unhashable)
# Tuple bisa karena immutable (hashable)

locations = {
    (35.6895, 139.6917): "Tokyo",
    (40.7128, -74.0060): "New York",
    (51.5074, -0.1278): "London"
}

print(locations[(35.6895, 139.6917)])  # Tokyo
```
