# Modules and Packages
**ID**: `M04-L05`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Organize code into modules, create packages with __init__.py, and use import statements correctly.

## Materi

### Konsep Dasar
**Module**: File Python (.py) yang berisi kode (functions, classes, variables). Setiap file adalah module.
**Package**: Folder yang berisi module-module atau sub-packages, ditandai dengan file `__init__.py` (bisa kosong).

**Import Statements**:
```python
import module_name
import package.module_name
from module_name import name1, name2
from package.module_name import something
from module_name import *
import module_name as alias
```

### Import Mechanics
- `sys.path`: list direktori di mana Python mencari module
- Current directory biasanya pertama dalam `sys.path`
- Package relative imports: `from . import sibling` (dalam package)
- `__init__.py`: executed saat package di-import, bisa define `__all__` untuk `from package import *`

### __name__ dan __main__
- `__name__` variable: jika file dieksekusi langsung, nilainya `"__main__"`; jika di-import, nilainya nama module
- Pattern `if __name__ == '__main__':` untuk kode yang hanya jalan saat script langsung di-run

### Detail Teknis
- Module adalah objek; atributnya bisa di-access via `module.attr`
- `dir(module)` menunjukkan apa yang diekspor
- `__all__` dalam module atau `__init__.py` mengontrol `from module import *`
- Absolute import (dari root package) lebih disarankan daripada relative
- Circular imports bisa menyebabkan error; hindari dengan design ulang
- Module di-cache di `sys.modules` setelah import pertama

## Latihan

### Soal 1
Buat struktur package sederhana:
```
myutils/
    __init__.py
    math_ops.py
    string_ops.py
```
- Di `math_ops.py`: fungsi `add(a, b)` dan `multiply(a, b)`
- Di `string_ops.py`: fungsi `reverse(s)` dan `capitalize_words(s)`
- Di `__init__.py`: import fungsi-fungsi tersebut agar bisa diakses langsung dari `myutils`
- Di script utama: `from myutils import add, reverse` dan gunakan.

Buat file-file tersebut dan buktikan importnya berjalan.

```python
# File: latihan_1_main.py
def latihan_1():
    from myutils import add, reverse
    print(add(2, 3))
    print(reverse("hello"))

if __name__ == "__main__":
    latihan_1()
```

**Expected Output**:
```
5
olleh
```

### Soal 2
Buat module `calculator.py` yang berisi fungsi `add`, `subtract`, `multiply`, `divide`. Di file terpisah, import module tersebut dan gunakan semua fungsi. Juga tunjukkan cara `import calculator as calc`.

```python
# File: latihan_2_main.py
def latihan_2():
    import calculator as calc
    print(calc.add(10, 5))
    print(calc.subtract(10, 5))
    print(calc.multiply(10, 5))
    print(calc.divide(10, 5))

if __name__ == "__main__":
    latihan_2()
```

**Expected Output**:
```
15
5
50
2.0
```
