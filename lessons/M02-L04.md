## Tujuan Pembelajaran
Use structural pattern matching for complex conditional logic.

## Materi

### Konsep Dasar
Match-case (structural pattern matching) diperkenalkan di Python 3.10. Mirip switch-case di bahasa lain, tapi lebih powerful karena bisa match berdasarkan struktur data, tidak hanya nilai.

Sintaks:
```python
match subject:
    case pattern1:
        # kode
    case pattern2 if guard:
        # kode dengan kondisi (guard)
    case _:
        # default (catch-all)
```

### Pattern Matching
- **Literal patterns**: match nilai literal (angka, string, boolean)
- **Name patterns**: capture value ke variable (seperti assignment)
- **Sequence patterns**: match list/tuple dengan nested patterns
- **Mapping patterns**: match dictionary dengan key patterns
- **Class patterns**: match instance dengan attributes
- **Or patterns**: `case 1 | 2 | 3:` (multiple alternatives)
- **Guard clauses**: `if condition` setelah pattern

### Detail Teknis
- `case _:` adalah wildcard (match apa pun, biasanya di akhir)
- Patterns dievaluasi berurutan, pertama yang match dieksekusi
- Bisa capture value dengan pattern variable (contoh: `case x:` mencaptur subject ke variable x)
- Sequence patterns bisa dengan `*rest` untuk capture sisa
- Mapping patterns hanya match keys yang ada, bisa dengan `**rest` untuk sisa
- Class patterns cocok untuk matching tipe data dan atributnya
- Performance: match-case sebenarnya if-elif-else dengan lebih banyak fitur
