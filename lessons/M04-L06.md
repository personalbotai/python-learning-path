# The if __name__ == '__main__' Pattern
**ID**: `M04-L06`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Make files executable as scripts and importable as modules.

## Materi

### Konsep Dasar
Setiap Python module memiliki built-in variable `__name__`. Nilainya tergantung bagaimana file dieksekusi:
- Jika dijalankan langsung (`python file.py`), `__name__ == "__main__"`
- Jika di-import (`import file`), `__name__ == "nama_file"` (nama module)

**Pattern**:
```python
def main():
    # kode utama program
    pass

if __name__ == "__main__":
    main()
```
Atau sederhana:
```python
if __name__ == "__main__":
    # kode langsung
```

### Mengapa Penting?
- **Reusability**: File bisa di-import tanpa mengeksekusi kode utama
- **Testing**: Bisa menambahkan test cases di bawah `if __name__ == "__main__":`
- **Modularitas**: Memisahkan definisi (functions, classes) dari eksekusi
- **Best Practice**: Setiap script yang bisa di-run langsung sebaiknya pakai pattern ini

### Detail Teknis
- `__name__` adalah string, bisa dicek dengan `==`
- Bisa juga mengecek `if __name__ != "__main__":` untuk kode yang hanya jalan saat import (tapi jarang digunakan)
- `sys.argv` bisa diakses di dalam `main()` untuk command-line arguments
- Bukan hanya untuk script; juga untuk package dengan `__main__.py` (jalankan `python -m package`)
- Jika tidak ada `if __name__ == "__main__":`, semua kode level-module akan dieksekusi saat import

## Latihan

### Soal 1
Buat module `math_utils.py` yang berisi fungsi `factorial(n)` dan `is_prime(n)`. Di akhir file, tambahkan `if __name__ == "__main__":` yang menjalankan test sederhana untuk kedua fungsi (misal: factorial(5) harus 120, is_prime(7) harus True). Buktikan bahwa saat di-import, test tidak berjalan.

```python
# math_utils.py
def factorial(n):
    # Tulis kode kamu di sini
    pass

def is_prime(n):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    # Test cases
    print(factorial(5))
    print(is_prime(7))
```

**Expected Output** (saat di-run langsung):
```
120
True
```
Saat di-import, tidak ada output.

### Soal 2
Buat script `greet.py` yang menerima nama dari command-line argument (`sys.argv[1]`) dan mencetak "Hello, <nama>!". Gunakan `if __name__ == "__main__":` dan pastikan file bisa di-import tanpa error (meski tidak berguna saat import).

```python
# greet.py
import sys

def greet(name):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    if len(sys.argv) > 1:
        greet(sys.argv[1])
    else:
        print("Usage: python greet.py <name>")
```

**Expected Output** (run: `python greet.py Alice`):
```
Hello, Alice!
```
