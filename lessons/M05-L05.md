## Materi

### Konsep Dasar

**Module**: Satu file Python (`.py`) yang berisi kode (functions, classes, variables).  
**Package**: Direktori yang berisi module-module (dan bisa sub-packages), harus ada `__init__.py`.  
**Module**: Semua kode yang bisa di-import.

**Struktur contoh:**

### Detail Teknis

**1. Import Statements**
```python
# Import entire module
import utils
result = utils.some_function()

# Import dengan alias
import utils as u
result = u.some_function()

# Import specific items
from utils import some_function, some_class
result = some_function()

# Import semua items (tidak disarankan karena namespace pollution)
from utils import *
result = some_function()  # tapi tidak jelas dari mana

# Import nested package
from services.auth import login
from models.user import User
```

**2. `__init__.py`**
```python
# File kosong: menandakan direktori sebagai package
# Bisa juga berisi code yang dieksekusi saat package di-import
# dan mendefinisikan __all__ untuk from package import *

# Contoh __init__.py di package `mypackage`
"""
MyPackage - contoh package
"""

__version__ = "1.0.0"
__author__ = "Alice"

# Import submodules agar mudah akses
from .module1 import func1
from .module2 import Class2

__all__ = ["func1", "Class2"]  # what gets imported with *
```

**3. Relative vs Absolute Imports**
```python
# Absolute imports (recommended)
from mypackage.module1 import something
from mypackage.subpackage.module2 import something_else

# Relative imports (dalam package)
# Gunakan . untuk current package, .. untuk parent
from .module1 import something        # same package
from ..module2 import something_else  # parent package
from ...subpackage.module import x    # grandparent package

# Hanya berlaku dalam package (jika script dijalankan langsung, error)
```

**4. Executable Modules with `__main__`**
```python
# mymodule.py
def main():
    print("Hello from mymodule!")

if __name__ == "__main__":
    # Code di sini hanya jalan jika file dieksekusi langsung:
    # python mymodule.py
    # Tidak jalan jika di-import: import mymodule
    main()
```

**5. Python Module Search Path (sys.path)**
```python
import sys
print(sys.path)
# List of directories Python cari module:
# - Current working directory
# - PYTHONPATH environment variable
# - Standard library directories
# - Site-packages (third-party)

# Tambah custom path
sys.path.append("/my/custom/path")
```

**6. Packages dengan Subpackages**
```
mypackage/
в”њв”Ђв”Ђ __init__.py
в”њв”Ђв”Ђ module1.py
в”њв”Ђв”Ђ module2.py
в””в”Ђв”Ђ subpackage/
    в”њв”Ђв”Ђ __init__.py
    в””в”Ђв”Ђ submodule.py
```

```python
# Import dari subpackage
from mypackage.subpackage.submodule import some_func

# __init__.py bisa expose API
# Di mypackage/subpackage/__init__.py:
from .submodule import some_func
__all__ = ["some_func"]

# Maka bisa:
from mypackage.subpackage import some_func
```

**7. Namespace Packages (PEP 420)**
```python
# Packages tanpa __init__.py (Python 3.3+)
# Berguna untuk split package across multiple locations
# Contoh: package `google` bisa di-install via `google-cloud-*` packages

# Struktur:
# /path1/google/__init__.py  (kosong atau tidak ada)
# /path2/google/cloud/storage.py

# Import: from google.cloud import storage
# Python merge semua `google` directories di sys.path
```

**8. Circular Imports (hati-hati!)**
```python
# module_a.py
from module_b import func_b  # import di top-level
def func_a():
    return "A"

# module_b.py
from module_a import func_a  # circular! bisa error
def func_b():
    return "B"

# Solusi:
# - Pindah import ke dalam fungsi (lazy import)
# - Refactor untuk pisah dependencies
# - Gunakan import di bottom (setelah definitions)
```

**9. `__all__` - kontrol `from module import *`**
```python
# mymodule.py
__all__ = ["public_func", "PublicClass"]  # only these imported with *

def public_func():
    pass

def _private_func():  # convention: underscore = private
    pass

class PublicClass:
    pass

class _PrivateClass:
    pass

# from mymodule import * hanya import public_func dan PublicClass
```

**10. Module Built-in Variables**
```python
print(__name__)   # "__main__" jika di-run langsung, else module name
print(__file__)   # path file module
print(__package__)  # package name jika ada, else None
print(__doc__)    # docstring module (jika ada)
```

### Soal 1: Create a Calculator Package

Struktur project:
```
calculator/
в”њв”Ђв”Ђ __init__.py
в”њв”Ђв”Ђ operations.py    # berisi add, subtract, multiply, divide
в”њв”Ђв”Ђ advanced.py      # berisi power, sqrt, factorial
в””в”Ђв”Ђ utils.py         # berisi helper functions
```

**operations.py:**
```python
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        raise ValueError("Division by zero")
    return a / b
```

**advanced.py:**
```python
import math

def power(base, exp):
    return base ** exp

def sqrt(x):
    if x < 0:
        raise ValueError("Cannot sqrt negative")
    return math.sqrt(x)

def factorial(n):
    if n < 0:
        raise ValueError("Factorial negative")
    if n <= 1:
        return 1
    return n * factorial(n-1)
```

**utils.py:**
```python
def is_number(s):
    try:
        float(s)
        return True
    except ValueError:
        return False
```

**__init__.py:** (expose API)
```python
from .operations import add, subtract, multiply, divide
from .advanced import power, sqrt, factorial
from .utils import is_number

__all__ = ["add", "subtract", "multiply", "divide",
           "power", "sqrt", "factorial", "is_number"]
__version__ = "1.0.0"
```

**Task:**
1. Buat struktur package di atas
2. Buat file `demo.py` di luar package yang menggunakan package:
```python
from calculator import add, multiply, factorial

print(add(5, 3))        # 8
print(multiply(4, 7))   # 28
print(factorial(5))     # 120
```
