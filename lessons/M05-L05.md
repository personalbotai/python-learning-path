# Properties, Getters, and Setters
**ID**: `M05-L05`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Implement computed properties with @property, create getter/setter decorators.

## Materi

### Konsep Dasar
Python menggunakan `@property` decorator untuk membuat computed attributes dan encapsulation tanpa perlu explicit getter/setter methods.

**Property**:
```python
class Celsius:
    def __init__(self, temperature):
        self._temperature = temperature

    @property
    def fahrenheit(self):
        return (self._temperature * 9/5) + 32

    @fahrenheit.setter
    def fahrenheit(self, value):
        self._temperature = (value - 32) * 5/9
```
- `@property` membuat method menjadi getter yang bisa diakses seperti attribute
- `@name.setter` membuat setter untuk property yang sama
- `@name.deleter` untuk delete (opsional)

### Getters and Setters
- Getter: method yang mengembalikan nilai attribute, diakses tanpa parentheses
- Setter: method yang menetapkan nilai attribute, dipanggil dengan assignment
- Decorator `@property` mengubah method menjadi attribute-like

### Detail Teknis
- Property bersifat read-only jika tidak ada setter
- Setter harus memiliki nama yang sama dengan property
- Deleter: `@name.deleter` untuk `del obj.name`
- Property bisa juga untuk computed attributes yang tidak tersimpan (hanya dihitung)
- Bisa menggunakan `@property` pada method tanpa parameter (selain self)
- Cocok untuk validation saat set, lazy computation, backward compatibility

## Latihan

### Soal 1
Buat class `Temperature` dengan:
- Private attribute `_celsius` (float)
- Property `celsius` (getter & setter) yang mengakses `_celsius`
- Property `fahrenheit` (hanya getter) yang mengembalikan nilai dalam Fahrenheit: `(celsius * 9/5) + 32`
- Setter untuk `celsius` harus memastikan nilai adalah number (gunakan `isinstance(value, (int, float))`) dan tidak None; jika invalid, raise `ValueError("Invalid temperature")`

```python
class Temperature:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    t = Temperature(25)
    print(t.celsius)      # 25
    print(t.fahrenheit)   # 77.0
    t.celsius = -40
    print(t.fahrenheit)   # -40.0
    # t.celsius = "hot"  # ValueError
```

**Expected Output**:
```
25
77.0
-40.0
```

### Soal 2
Buat class `Circle` dengan:
- `__init__(self, radius)`
- Property `radius` (getter & setter) dengan validation: radius harus >= 0, else ValueError
- Property `diameter` (hanya getter) → `2 * radius`
- Property `area` (hanya getter) → `π * radius²` (gunakan `math.pi`)
- Setter untuk `radius` hanya mengubah `_radius`, `diameter` dan `area` otomatis recompute

```python
import math

class Circle:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    c = Circle(5)
    print(c.radius)    # 5
    print(c.diameter)  # 10
    print(round(c.area, 2))  # 78.54
    c.radius = 10
    print(c.diameter)  # 20
```

**Expected Output**:
```
5
10
78.54
20
```
