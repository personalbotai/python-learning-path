## Materi

### Konsep Dasar

Statement `if` digunakan untuk mengambil keputusan berdasarkan kondisi. Python menggunakan **indentation** (spasi/tab) untuk menentukan blok kode.

**Struktur dasar:**
```python
if kondisi:
    # blok kode dijalankan jika kondisi True
    print("Kondisi terpenuhi")
else:
    # blok kode dijalankan jika kondisi False
    print("Kondisi tidak terpenuhi")
```

**Dengan `elif` (else if):**
```python
if nilai >= 90:
    grade = "A"
elif nilai >= 80:
    grade = "B"
elif nilai >= 70:
    grade = "C"
elif nilai >= 60:
    grade = "D"
else:
    grade = "F"
```

**Contoh lengkap:**
```python
def nilai_ke_huruf(nilai):
    if nilai >= 90:
        return "A"
    elif nilai >= 80:
        return "B"
    elif nilai >= 70:
        return "C"
    elif nilai >= 60:
        return "D"
    else:
        return "F"

print(nilai_ke_huruf(85))  # B
```

### Detail Teknis

**1. Indentation adalah SEBAGIAN dari syntax**
```python
# SALAH - missing indentation
if x > 0:
print("Positive")  # IndentationError!

# BENAR
if x > 0:
    print("Positive")
```

**2. Multiple conditions dengan `and`, `or`, `not`**
```python
usia = 25
status_pekerjaan = "employed"

if usia >= 18 and status_pekerjaan == "employed":
    print("Dewasa dan bekerja")

if usia < 18 or status_pekerjaan == "student":
    print("Anak atau pelajar")

if not status_pekerjaan == "unemployed":
    print("Memiliki pekerjaan")
```

**3. Nested if (if di dalam if)**
```python
nilai = 85
if nilai >= 60:
    if nilai >= 80:
        print("Lulus dengan baik")
    else:
        print("Lulus")
else:
    print("Tidak lulus")
```

**4. Ternary Operator (one-liner if)**
```python
# Syntax: nilai_if_true if condition else nilai_if_false
usia = 20
status = "dewasa" if usia >= 17 else "anak"
print(status)  # dewasa

# Contoh lain
x = 10
y = 20
max_val = x if x > y else y
```

**5. Pattern Matching dengan `match` (Python 3.10+)**
```python
status = "success"

match status:
    case "success":
        print("Operasi berhasil")
    case "error":
        print("Terjadi error")
    case "pending":
        print("Menunggu")
    case _:
        print("Status tidak dikenal")
```
