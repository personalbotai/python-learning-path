# Basic Input/Output and String Formatting
**ID**: `M01-L04`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Use print() and input(), format strings with f-strings, .format(), and % operator.

## Materi

### Konsep Dasar
Fungsi `print()` digunakan untuk menampilkan output ke konsol. Bisa menerima beberapa argumen dan secara otomatis menambahkan spasi di antara mereka. `input()` digunakan untuk menerima input dari pengguna, akan selalu mengembalikan string walaupun pengguna mengetik angka.

String formatting memungkinkan kita menyisipkan nilai ke dalam string. Ada beberapa cara:
- f-strings (Python 3.6+): `f"Nama: {name}, Umur: {age}"`
- .format(): `"Nama: {}, Umur: {}".format(name, age)`
- % operator (lama): `"Nama: %s, Umur: %d" % (name, age)`

f-strings adalah cara terbaru dan paling readable karena expression langsung ditulis dalam kurung kurawal.

### Detail Teknis
- `print()` parameter `sep` untuk separator, `end` untuk akhiri baris
- `input(prompt)` menampilkan prompt sebelum menerima input
- f-strings bisa mengandung expression termasuk pemanggilan fungsi
- Format specifier seperti `{value:.2f}` untuk 2 desimal, `{value:>10}` untuk rata kanan

## Latihan

### Soal 1
Buat program yang meminta nama dan umur pengguna, lalu menampilkan sapaan dengan format: "Halo [nama]! Usia kamu [umur] tahun."

```python
def latihan_1():
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    latihan_1()
```

**Expected Output**:
```
Masukkan nama: Alice
Masukkan umur: 25
Halo Alice! Usia kamu 25 tahun.
```

### Soal 2
Buat fungsi yang menerima angka (integer atau float) dan mengembalikan string dengan format: "Angka: [nilai] (bulat: [bulat], desimal: [desimal])". Gunakan f-string dengan format specifier.

```python
def latihan_2(angka):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    print(latihan_2(42))
    print(latihan_2(3.14159))
```

**Expected Output**:
```
Angka: 42 (bulat: 42, desimal: 42.00)
Angka: 3.14159 (bulat: 3, desimal: 3.14)
```
