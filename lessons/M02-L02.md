## Materi

### Konsep Dasar

`while` loop akan terus berjalan selama kondisi Bernama (`True`). Berguna ketika jumlah iterasi tidak diketahui sebelumnya.

**Syntax dasar:**
```python
while kondisi:
    # lakukan sesuatu
    # pastikan ada perubahan yang membuat kondisi eventually False
```

**Contoh sederhana:**
```python
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1  # WAJUB: update kondisi, else infinite loop!
```

### Detail Teknis

**1. While loop dengan input user**
```python
# Loop sampai user memasukkan 'quit'
command = ""
while command != "quit":
    command = input("Masukkan command (atau 'quit' untuk keluar): ")
    if command == "hello":
        print("Halo!")
    elif command == "time":
        import datetime
        print(f"Sekarang jam {datetime.datetime.now().hour}")
```

**2. Infinite loop dengan `while True`**
```python
while True:
    user_input = input("Masukkan perintah: ")
    if user_input == "exit":
        print("Keluar...")
        break  # keluar dari loop
    elif user_input == "skip":
        continue  # lanjut ke iterasi berikutnya
    else:
        print(f"Kamu memasukkan: {user_input}")
```

**3. `break` - keluar sepenuhnya dari loop**
```python
# Cari angka pertama yang habis dibagi 7
for i in range(1, 101):
    if i % 7 == 0:
        print(f"Angka pertama yang habis dibagi 7: {i}")
        break  # keluar dari for loop
```

**4. `continue` - skip ke iterasi berikutnya**
```python
# Print hanya angka genap
for i in range(1, 11):
    if i % 2 != 0:
        continue  # skip angka ganjil
    print(i)  # hanya angka genap yang sampai sini: 2,4,6,8,10
```

**5. `else` clause pada while (jarang digunakan)**
```python
# else dieksekusi jika while selesai tanpa break
attempt = 0
while attempt < 3:
    password = input("Password: ")
    if password == "secret":
        print("Login berhasil!")
        break
    attempt += 1
else:
    print("3x gagal. Account terkunci.")  # hanya jika tidak ada break
```

**6. While vs For - kapan pakai yang mana?**

Gunakan **`for`** quando:
- Anda tahu jumlah iterasi sebelumnya (misal: loop melalui list, range, dll)
- Iterasi atas sequence tertentu

Gunakan **`while`** quando:
- Jumlah iterasi tidak diketahui (tergantung kondisi runtime)
- Loop sampai suatu kondisi terpenuhi (misal: user input tertentu, file EOF, dll)
