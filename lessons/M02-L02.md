## Tujuan Pembelajaran
Use while loops and for loops, understand range(), and loop control (break, continue, else).

## Materi

### Konsep Dasar
Loop digunakan untuk mengulang eksekusi kode. Python memiliki dua jenis loop utama:

**while loop**: Mengulang selama kondisi bernilai True.
```python
while kondisi:
    # kode di sini diulang
```

**for loop**: Mengulang untuk setiap elemen dalam iterable (list, tuple, string, range, dll).
```python
for item in iterable:
    # kode di sini diulang untuk setiap item
```

**range()**: Fungsi untuk menghasilkan sequence angka.
- `range(stop)`: 0 sampai stop-1
- `range(start, stop)`: start sampai stop-1
- `range(start, stop, step)`: dengan langkah step

### Loop Control Statements
- `break`: Keluar dari loop segera
- `continue`: Lewati sisa iterasi saat ini, lanjut ke iterasi berikutnya
- `else`: Blok else setelah loop hanya dieksekusi jika loop selesai normally (tidak via break)

### Detail Teknis
- while loop bisa infinite jika kondisi selalu True (butuh break untuk keluar)
- for loop menggunakan iterator protocol di balik layar
- range() menghasilkan range object (lazy), bukan list (tapi bisa di-cast ke list)
- else pada loop berguna untuk pencarian: jika tidak ditemukan (loop selesai tanpa break)
- Perlu hati-hati dengan off-by-one errors saat menggunakan range
