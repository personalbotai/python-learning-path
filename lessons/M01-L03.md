## Tujuan Pembelajaran
Learn about dynamic typing, basic types (int, float, str, bool), type conversion, and type hints introduction.

## Materi

### Konsep Dasar
Python menggunakan dynamic typing, artinya tipe data variabel ditentukan secara otomatis berdasarkan nilai yang diberikan, tanpa deklarasi eksplisit. Tipe data dasar di Python meliputi:
- `int`: bilangan bulat (contoh: 42, -10)
- `float`: bilangan desimal (contoh: 3.14, -0.5)
- `str`: teks/string (contoh: "hello", 'world')
- `bool`: boolean True atau False

Type conversion dapat dilakukan dengan fungsi seperti `int()`, `float()`, `str()`, `bool()`. Namun, perlu hati-hati karena konversi yang tidak sesuai dapat menyebabkan error (misal: int("abc")). Python 3.5+ mendukung type hints yang memungkinkan menambahkan anotasi tipe untuk meningkatkan readability dan tooling support, meski tidak memaksa.

### Detail Teknis
- Gunakan `type(variable)` untuk memeriksa tipe data
- Type hints hanya informatif, tidak mengharuskan type checking saat runtime
- Konversi string ke number harus memastikan string tersebut valid
- Python memiliki tipe lain seperti `None`, `list`, `dict`, dll

## Latihan

### Soal 1
Buat variabel dengan tipe data yang berbeda (int, float, str, bool) dan tampilkan tipe masing-masing menggunakan fungsi `type()`.

```python
def latihan_1():
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    latihan_1()
```

**Expected Output**:
```
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
```

### Soal 2
Buat fungsi yang menerima dua angka (string atau number) dan mengembalikan hasil penjumlahan dengan konversi tipe yang tepat. Jika input berupa string, convert ke float terlebih dahulu.

```python
def latihan_2(a, b):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    print(latihan_2(5, 3))        # Output: 8
    print(latihan_2("5.5", 2.5))  # Output: 8.0
    print(latihan_2("10", "20")) # Output: 30
```

**Expected Output**:
```
8
8.0
30
```
