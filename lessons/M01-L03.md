## Materi

### Konsep Dasar

Operator adalah simbol yang melakukan operasi pada operand (nilai/variabel).

**1. Operator Aritmatika**
```python
a = 10
b = 3

# + : penjumlahan
print(a + b)   # 13

# - : pengurangan
print(a - b)   # 7

# * : perkalian
print(a * b)   # 30

# / : pembagian (selalu hasil float)
print(a / b)   # 3.3333333333333335

# // : floor division (pembagian integer)
print(a // b)  # 3

# % : modulus (sisa pembagian)
print(a % b)   # 1

# ** : pangkat
print(a ** b)  # 1000
```

**2. Operator Perbandingan (Comparison)**
```python
x = 5
y = 10

print(x == y)   # False (sama)
print(x != y)   # True  (tidak sama)
print(x > y)    # False
print(x < y)    # True
print(x >= y)   # False
print(x <= y)   # True
```

Hasil operator perbandingan selalu `True` atau `False` (boolean).

**3. Operator Logika (Boolean)**
```python
a = True
b = False

print(a and b)  # False (keduanya True)
print(a or b)   # True  (salah satu True)
print(not a)    # False (negasi)
```

**4. Operator Assignment (Penugasan)**
```python
x = 10        # assignment sederhana
x += 5        # x = x + 5  → 15
x -= 3        # x = x - 3  → 12
x *= 2        # x = x * 2  → 24
x /= 4        # x = x / 4  → 6.0
x //= 2       # x = x // 2 → 3.0
x %= 2        # x = x % 2  → 1.0
x **= 3       # x = x ** 3 → 1.0
```

**5. Operator Identitas & Keanggotaan**
```python
# Identity (adalah vs sama dengan)
a = [1, 2, 3]
b = a          # b refer ke objek yang sama
c = [1, 2, 3]  # c adalah objek baru

print(a is b)   # True  (sama object)
print(a is c)   # False (objek berbeda)
print(a == c)   # True  (nilai sama)

# Membership (apakah ada di dalam)
numbers = [1, 2, 3, 4, 5]
print(3 in numbers)      # True
print(6 not in numbers)  # True
```

### Detail Teknis

**Operator Precedence (urutan eksekusi):**
```python
# Precedence tinggi ke rendah:
# ** (pangkat)
# +x, -x, ~x (unary)
# *, /, //, %
# +, -
# <, <=, >, >=
# ==, !=
# not
# and
# or

# Contoh:
result = 2 + 3 * 4
# 3 * 4 = 12 (dulu)
# 2 + 12 = 14
print(result)  # 14

# Gunakan parentheses untuk jelas:
result = (2 + 3) * 4  # 5 * 4 = 20
```

**Short-circuit Evaluation:**
```python
# and: jika operand pertama False, operand kedua tidak dievaluasi
def hello():
    print("hello called")
    return True

def world():
    print("world called")
    return False

False and hello()  # hello() tidak dipanggil
hello() and world()  # world() dipanggil karena hello() True

# or: jika operand pertama True, operand kedua tidak dievaluasi
True or hello()  # hello() tidak dipanggil
False or world()  # world() dipanggil
```
