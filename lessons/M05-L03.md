## Materi

### Konsep Dasar

**Lambda Functions** adalah anonymous functions (tanpa nama) dengan sintaks singkat. Digunakan untuk fungsi sederhana yang sekali pakai.

**Basic Lambda:**
```python
# Regular function
def square(x):
    return x ** 2

# Lambda equivalent
square = lambda x: x ** 2
print(square(5))  # 25

# Lambda langsung (tanpa assignment)
(lambda x: x ** 2)(5)  # 25

# Multiple arguments
add = lambda a, b: a + b
print(add(3, 4))  # 7
```

**Higher-Order Functions (HOF)** adalah fungsi yang:
1. Menerima fungsi lain sebagai argument
2. Mengembalikan fungsi sebagai result
3. Atau keduanya

### Detail Teknis

**1. Built-in Higher-Order Functions**

```python
# map(function, iterable) - transform setiap elemen
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, numbers))  # [1, 4, 9, 16, 25]

# Bisa dengan regular function
def to_upper(s):
    return s.upper()

words = ["hello", "world"]
upper_words = list(map(to_upper, words))  # ['HELLO', 'WORLD']

# Multiple iterables
a = [1, 2, 3]
b = [4, 5, 6]
sums = list(map(lambda x, y: x + y, a, b))  # [5, 7, 9]

# filter(function, iterable) - filter elemen berdasarkan kondisi
numbers = [1, 2, 3, 4, 5, 6]
evens = list(filter(lambda x: x % 2 == 0, numbers))  # [2, 4, 6]

# filter dengan None untuk menghapus falsy values
mixed = [0, 1, "", "hello", None, [], [1,2]]
truthy = list(filter(None, mixed))  # [1, 'hello', [1,2]]

# reduce(function, iterable[, initializer]) - kumulatif reduce ke satu nilai
from functools import reduce
numbers = [1, 2, 3, 4, 5]
product = reduce(lambda acc, x: acc * x, numbers)  # 120 (1*2*3*4*5)
# with initializer
product = reduce(lambda acc, x: acc * x, numbers, 10)  # 1200 (10*1*2*3*4*5)

# reduce untuk concat strings
words = ["Hello", "World", "Python"]
sentence = reduce(lambda a, b: a + " " + b, words)  # "Hello World Python"
```

**2. Custom Higher-Order Functions**

```python
# Function yang return function (closure)
def make_multiplier(factor):
    """Return function yang multiply argument dengan factor"""
    def multiplier(x):
        return x * factor
    return multiplier

double = make_multiplier(2)
triple = make_multiplier(3)

print(double(5))   # 10
print(triple(5))   # 15

# Closure: inner function "captures" variable dari outer scope
def make_greeting(prefix):
    def greet(name):
        return f"{prefix} {name}"
    return greet

say_hello = make_greeting("Hello")
say_hi = make_greeting("Hi")
print(say_hello("Alice"))  # "Hello Alice"
print(say_hi("Bob"))       # "Hi Bob"
```

**3. Closures & Lexical Scoping**

```python
def outer(x):
    def inner(y):
        return x + y  # inner bisa akses x dari outer scope
    return inner

add5 = outer(5)
print(add5(3))  # 8
print(add5(10)) # 15

# x tetap di-"remember" oleh closure
# Ini berbeda dengan variabel lokal biasa
```

**4. Decorators (special HOF)**

```python
# Simple decorator
def my_decorator(func):
    def wrapper():
        print("Sebelum fungsi dipanggil")
        result = func()
        print("Setelah fungsi dipanggil")
        return result
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
# Output:
# Sebelum fungsi dipanggil
# Hello!
# Setelah fungsi dipanggil

# Decorator dengan arguments
def repeat(n):
    def decorator(func):
        def wrapper(*args, **kwargs):
            results = []
            for _ in range(n):
                result = func(*args, **kwargs)
                results.append(result)
            return results
        return wrapper
    return decorator

@repeat(times=3)
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # ['Hello, Alice!', 'Hello, Alice!', 'Hello, Alice!']
```

**5. functools Module**

```python
from functools import partial, wraps

# partial: freeze some arguments
def power(base, exponent):
    return base ** exponent

square = partial(power, exponent=2)
cube = partial(power, exponent=3)

print(square(5))  # 25
print(cube(3))    # 27

# wraps: preserve metadata saat decorator
def my_decorator(func):
    @wraps(func)  # penting: preserve __name__, __doc__, dll
    def wrapper(*args, **kwargs):
        """Wrapper docstring"""
        return func(*args, **kwargs)
    return wrapper

@my_decorator
def add(a, b):
    """Add two numbers"""
    return a + b

print(add.__name__)   # "add" (bukan "wrapper")
print(add.__doc__)    # "Add two numbers" (bukan "Wrapper docstring")
```

**6. Function Composition**

```python
# compose(f, g)(x) = f(g(x))
def compose(f, g):
    return lambda x: f(g(x))

def double(x):
    return x * 2

def increment(x):
    return x + 1

double_then_increment = compose(increment, double)
print(double_then_increment(5))  # (5*2)+1 = 11

increment_then_double = compose(double, increment)
print(increment_then_double(5))  # (5+1)*2 = 12
```

**7. Memoization (caching) dengan closures**

```python
def memoize(func):
    cache = {}
    def wrapper(*args):
        if args in cache:
            return cache[args]
        result = func(*args)
        cache[args] = result
        return result
    return wrapper

@memoize
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# fibonacci(35) akan sangat cepat karena cache
```

```python
    """

    - 'positive': filter angka > 0
    - 'even': filter angka genap
    - 'multiple_of': return function yang filter kelipatan n
    - 'length_greater_than': return function yang filter string dengan panjang > n
    """
    pass

def main():
    # Positive filter
    positive_filter = make_filter('positive')
    print(positive_filter([-1, 0, 1, 2, -5]))  # [1, 2]

    # Even filter
    even_filter = make_filter('even')
    print(even_filter([1,2,3,4,5]))  # [2,4]

    # Multiple of filter
    mult5 = make_filter('multiple_of')
    mult5 = make_filter('multiple_of')(5)  # atau cara lain
    print(mult5([3,5,10,12,15]))  # [5,10,15]

    # Length filter
    long_filter = make_filter('length_greater_than')(3)
    print(long_filter(["cat", "dog", "elephant", "bird"]))  # ["elephant"]
```
