# Inheritance and Method Resolution Order (MRO)
**ID**: `M05-L03`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Create subclass relationships, use super(), understand multiple inheritance and MRO.

## Materi

### Konsep Dasar
**Inheritance**: Subclass mewarisi atribut dan methods dari superclass. Memungkinkan code reuse dan polymorphism.

**Syntax**:
```python
class SubClass(SuperClass):
    # override atau tambah methods
```

**super()**: Memanggil method dari superclass.
```python
class Sub(Super):
    def method(self):
        super().method()  # panggil Super.method
```

**Multiple Inheritance**: Subclass bisa memiliki lebih dari satu superclass.
```python
class Child(Parent1, Parent2):
    pass
```

**Method Resolution Order (MRO)**: Urutan di mana Python mencari method saat multiple inheritance. MRO ditentukan oleh C3 linearization algorithm. Dapat dilihat dengan `ClassName.__mro__` atau `ClassName.mro()`.

### Detail Teknis
- `super()` tanpa argumen dalam instance method otomatis menggunakan class instance dan instance itu sendiri
- Di Python 3, `super()` bisa dipakai tanpa arguments dalam kebanyakan kasus
- MRO memastikan setiap class dicari tepat sekali (no duplication) dan urutan konsisten
- Diamond problem (sebuah class diwariskan dua kali) diselesaikan oleh MRO
- `super()` bisa digunakan di `__init__` untuk memanggil `super().__init__()`
- Bisa akses superclass langsung dengan `SuperClass.method(self, ...)` tapi `super()` lebih fleksibel untuk multiple inheritance

## Latihan

### Soal 1
Buat class hierarchy:
- `Animal` dengan `__init__(self, name)` dan method `speak(self)` yang mengembalikan `"..."` (placeholder)
- `Dog(Animal)` yang override `speak()` mengembalikan `"Woof!"`
- `Cat(Animal)` yang override `speak()` mengembalikan `"Meow!"`
- Test: buat Dog dan Cat, panggil speak().

```python
class Animal:
    # Tulis kode kamu di sini
    pass

class Dog(Animal):
    # Tulis kode kamu di sini
    pass

class Cat(Animal):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    dog = Dog("Buddy")
    cat = Cat("Whiskers")
    print(dog.speak())  # Woof!
    print(cat.speak())  # Meow!
```

**Expected Output**:
```
Woof!
Meow!
```

### Soal 2
Buat multiple inheritance:
- `A` dengan method `foo()` yang print "A.foo"
- `B` dengan method `foo()` yang print "B.foo"
- `C(A, B)` yang override `foo()` dan memanggil `super().foo()` (harus print "A.foo" lalu "B.foo" sesuai MRO)
- Cek MRO dengan `C.__mro__` atau `C.mro()`

```python
class A:
    def foo(self):
        print("A.foo")

class B:
    def foo(self):
        print("B.foo")

class C(A, B):
    def foo(self):
        super().foo()
        B.foo(self)  # eksplisit

if __name__ == "__main__":
    c = C()
    c.foo()
    print(C.mro())
```

**Expected Output**:
```
A.foo
B.foo
[<class '__main__.C'>, <class '__main__.A'>, <class '__main__.B'>, <class 'object'>]
```
