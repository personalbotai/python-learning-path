## Materi

### Konsep Dasar

File I/O adalah cara berinteraksi dengan filesystem. Python menggunakan built-in function `open()` untuk mendapatkan file object.

**Basic open/read/close:**
```python
# Buka file untuk dibaca
f = open("file.txt", "r")  # "r" = read mode (default)
content = f.read()         # baca seluruh file sebagai string
f.close()                  # penting: selalu close!

# With statement (recommended - auto close)
with open("file.txt", "r") as f:
    content = f.read()  # file otomatis close setelah block
```

**File modes:**
- `"r"` - read (default, error jika file tidak ada)
- `"w"` - write (truncate, create jika tidak ada)
- `"x"` - exclusive creation (error jika file sudah ada)
- `"a"` - append (tulis di akhir file)
- `"b"` - binary mode (e.g., `"rb"`, `"wb"`)
- `"t"` - text mode (default)
- `"+"` - update (read + write), e.g., `"r+"`, `"w+"`

### Detail Teknis

**1. Reading Files**

```python
# Read entire file as string
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# Read line by line (memory efficient untuk file besar)
with open("file.txt", "r") as f:
    for line in f:  # iterate langsung, tiap line ada newline
        print(line.strip())  # strip() hapus newline & spaces

# Read all lines into list
with open("file.txt") as f:
    lines = f.readlines()  # list of strings, tiap line termasuk newline

# Read specific number of characters
with open("file.txt") as f:
    first_100 = f.read(100)  # baca 100 karakter pertama
    # Posisi file pointer maju 100 chars
    next_chars = f.read(50)  # lanjut dari posisi sebelumnya

# Check file pointer position
with open("file.txt") as f:
    f.read(10)
    pos = f.tell()  # 10 (byte position)
    f.seek(0)       # kembali ke awal
    content = f.read()
```

**2. Writing Files**

```python
# Write string to file (overwrite)
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello, World!\n")
    f.write("Second line\n")

# Write multiple lines (list of strings)
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("output.txt", "w") as f:
    f.writelines(lines)  # tidak tambah newline otomatis

# Append to existing file
with open("output.txt", "a") as f:
    f.write("Appended line\n")

# Write with print() ke file (otomatis newline)
with open("output.txt", "w") as f:
    print("Hello", file=f)
    print("World", file=f)
```

**3. Encoding**

```python
# Default encoding depends on system (UTF-8 di Linux/macOS modern, cp1252 di Windows)
# Always specify encoding explicitly untuk portabilitas
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()

# Write dengan encoding tertentu
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Halo, 世界 🌍")  # Unicode characters

# List available encodings

```

**4. Newline Handling**

```python
# Text mode ("t" default): newline conversion
# - Saat read: \r\n, \r, \n semua dikonversi ke \n
# - Saat write: \n dikonversi ke platform-native newline (\n di Unix, \r\n di Windows)

# Binary mode ("b"): no conversion, raw bytes
with open("file.txt", "rb") as f:
    data = f.read()  # bytes object, newline tetap apa adanya
```

**5. Common Exceptions**

```python
try:
    with open("nonexistent.txt", "r") as f:
        content = f.read()
except FileNotFoundError:
    print("File tidak ditemukan")
except PermissionError:
    print("Tidak ada permission")
except IsADirectoryError:
    print("Itu direktori, bukan file")
except UnicodeDecodeError:
    print("Encoding error saat decode")
except OSError as e:  # base class untuk file errors
    print(f"OS error: {e}")
```

**6. File Operations**

```python
import os

# Check file exists
os.path.exists("file.txt")  # True/False

# Get file size
os.path.getsize("file.txt")  # bytes

# Read as bytes (binary)
with open("image.png", "rb") as f:
    data = f.read()  # bytes
    # Bisa write ke file lain
    with open("copy.png", "wb") as out:
        out.write(data)

# Seek dan tell untuk random access
with open("large.bin", "rb") as f:
    f.seek(1024)  # pindah ke byte 1024
    chunk = f.read(512)  # baca 512 bytes dari posisi itu
```
