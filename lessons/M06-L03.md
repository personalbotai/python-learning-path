# Context Managers and the with Statement
**ID**: `M06-L03`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Use with for resource management, create custom context managers with __enter__ and __exit__.

## Materi

### Konsep Dasar
Context managers mengelola resource (seperti file, lock, koneksi) dengan memastikan cleanup dilakukan meskipun ada exception. Digunakan dengan `with` statement.

**Built-in Context Managers**: `open()`, `threading.Lock()`, `sqlite3.connect()`, dll.

**Syntax**:
```python
with context_manager as variable:
    # block code
# otomatis cleanup (exit) terjadi di sini
```

### Custom Context Manager dengan __enter__ dan __exit__
Class dapat mengimplementasikan dua methods untuk menjadi context manager:
- `__enter__(self)`: dipanggil saat masuk ke `with` block, mengembalikan value yang akan di-assign ke `as variable`
- `__exit__(self, exc_type, exc_val, exc_tb)`: dipanggil saat keluar dari `with` block. Parameter exception info (jika ada exception). Jika return True, exception akan di-suppress.

### contextlib.contextmanager
Decorator untuk membuat context manager dengan generator function (lebih sederhana).
```python
from contextlib import contextmanager

@contextmanager
def my_context():
    # setup
    yield resource
    # cleanup
```

### Detail Teknis
- `__exit__` harus return boolean: `True` untuk suppress exception, `False` atau `None` untuk propagate
- `__enter__` bisa return apa pun (will be assigned to variable after `as`)
- `contextlib.contextmanager` generator harus yield tepat sekali; cleanup code setelah yield
- Context manager juga bisa digunakan tanpa `as` jika tidak perlu variable
- `with` bisa nested: `with a() as x, b() as y:`
- `contextlib.ExitStack` untuk multiple context managers dinamis

## Latihan

### Soal 1
Buat custom context manager `Timer` yang mengukur waktu eksekusi block.
- `__enter__` catat waktu awal (gunakan `time.time()`)
- `__exit__` catat waktu akhir, hitung durasi, print "Elapsed: X.XX seconds"
- Jangan suppress exceptions (return False dari `__exit__`)

```python
import time

class Timer:
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    with Timer():
        total = sum(range(1000000))
```

**Expected Output**:
```
Elapsed: 0.045 seconds (approx)
```

### Soal 2
Buat context manager `SuppressErrors` yang men-suppress exceptions tertentu (diberikan saat konstruksi). `__exit__` harus return True jika exception type cocok, agar exception tidak propagate.

```python
class SuppressErrors:
    def __init__(self, *exceptions):
        # Tulis kode kamu di sini
        pass
    
    def __enter__(self):
        pass
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        # Tulis kode kamu di sini
        pass

if __name__ == "__main__":
    with SuppressErrors(ZeroDivisionError, TypeError):
        result = 1 / 0  # akan di-suppress
        int("abc")      # TypeError juga di-suppress
    print("Continuing...")  # tetap dieksekusi
```

**Expected Output**:
```
Continuing...
```
