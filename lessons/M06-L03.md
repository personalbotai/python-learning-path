## Materi

### Konsep Dasar

`pathlib` (Python 3.4+) adalah modern way untuk bekerja dengan paths. Sebelumnya, kita pakai `os.path` yang string-based. `pathlib` menggunakan object-oriented approach dengan `Path` objects, lebih readable dan cross-platform (Windows/Unix).

**Basic Path creation:**
```python
from pathlib import Path

# Current directory
p = Path.cwd()
print(p)  # /home/user/project (Linux) atau C:\Users\user\project (Windows)

# Home directory
p = Path.home()

# Specific path
p = Path("/etc/passwd")  # absolut
p = Path("data/file.txt")  # relative
p = Path(".") / "subdir" / "file.txt"  # gunakan / operator untuk join
```

### Detail Teknis

**1. Path Properties & Methods**
```python
p = Path("/home/alice/documents/report.pdf")

# Properties
print(p.name)        # "report.pdf"
print(p.stem)        # "report" (filename without suffix)
print(p.suffix)      # ".pdf"
print(p.parent)      # /home/alice/documents
print(p.anchor)      # "/" (root)
print(p.drive)       # "C:" di Windows, "" di Unix

# Methods
print(p.exists())    # True jika file/directory ada
print(p.is_file())   # True jika regular file
print(p.is_dir())    # True jika directory
print(p.resolve())   # return absolute path (resolve symlinks)
```

**2. Creating & Removing**
```python
# Create directory
p = Path("new_folder")
p.mkdir()            # create single directory
p.mkdir(parents=True, exist_ok=True)  # create parents juga, tidak error jika sudah ada

# Create file (touch)
p = Path("empty.txt")
p.touch()            # create empty file, update timestamp jika sudah ada
p.touch(exist_ok=False)  # error jika sudah ada

# Remove
p.unlink()           # delete file
p.rmdir()            # delete empty directory
# For non-empty directory, gunakan shutil.rmtree()

# Rename / Move
p = Path("old.txt")
p.rename("new.txt")  # rename atau move ke path lain
```

**3. Reading & Writing Files dengan Path**
```python
# Path objects have read_text/write_text/read_bytes/write_bytes methods
p = Path("hello.txt")
p.write_text("Hello, World!\n", encoding="utf-8")  # write string
content = p.read_text(encoding="utf-8")  # read string

# Binary
p.write_bytes(b"binary data")
data = p.read_bytes()

# Append mode (tidak langsung disediakan, pakai open)
with p.open("a", encoding="utf-8") as f:
    f.write("Appended line\n")
```

**4. Globbing (Pattern Matching)**
```python
p = Path(".")

# *.txt files in current directory
for txt_file in p.glob("*.txt"):
    print(txt_file)

# Recursive: semua .py files di subdirectories
for py_file in p.rglob("*.py"):
    print(py_file)  # relative path dari p

# Pattern lain:
# "*.{jpg,png}"  - jpg atau png
# "**/*.py"      - recursive (sama dengan rglob)
# "[a-z]*.txt"   - dimulai dengan huruf kecil
```

**5. Directory Iteration**
```python
p = Path("some_dir")

# Iterate direktori (entries di dalamnya)
for item in p.iterdir():
    print(item.name, "dir" if item.is_dir() else "file")

# Filter dengan list comprehension
files = [f for f in p.iterdir() if f.is_file()]
txt_files = [f for f in p.rglob("*.txt")]  # recursive
```

**6. File Metadata**
```python
p = Path("file.txt")

stat = p.stat()  # os.stat_result object
print(stat.st_size)        # size in bytes
print(stat.st_mtime)       # modification time (timestamp)
from datetime import datetime
mtime = datetime.fromtimestamp(stat.st_mtime)
print(f"Modified: {mtime}")

# Shortcuts
print(p.stat().st_size)  # size
```

**7. shutil - High-level File Operations**
```python
import shutil

# Copy file
shutil.copy("source.txt", "destination.txt")  # copy file + permission mode
shutil.copy2("source.txt", "dest.txt")        # copy juga metadata (mtime, atime)
shutil.copytree("source_dir", "dest_dir")    # copy entire directory tree

# Move / rename
shutil.move("old.txt", "new.txt")
shutil.move("file.txt", "subdir/")  # move ke directory

# Delete directory tree
shutil.rmtree("temp_folder")  # hapus recursively

# Archive (zip, tar)
shutil.make_archive("backup", "zip", "folder_to_zip")
shutil.unpack_archive("backup.zip", "extract_to")
```

**8. Path Manipulation Lintas OS**
```python
# Path otomatis handle separator ( / vs \ )
p = Path("folder") / "subfolder" / "file.txt"
print(p)  # folder/subfolder/file.txt di Linux, folder\subfolder\file.txt di Windows

# Convert ke string sesuai OS
str(p)  # string dengan separator OS-native
p.as_posix()  # always forward slash (POSIX style)

# Parse existing path
p = Path("C:\\Users\\Alice\\file.txt")  # Windows path
print(p.drive)  # "C:"
print(p.parts)  # ('C:\\', 'Users', 'Alice', 'file.txt')
```
