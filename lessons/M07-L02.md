## Materi

### Konsep Dasar
**CSV Module**:
```python
import csv

# Reading
with open('file.csv', newline='') as f:
    reader = csv.reader(f)  # returns iterator of lists
    for row in reader:
        print(row)  # list of strings

# DictReader (more convenient)
with open('file.csv') as f:
    reader = csv.DictReader(f)  # each row is dict dengan keys dari header
    for row in reader:
        print(row['name'])

# Writing
with open('out.csv', 'w', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['name', 'age'])  # header
    writer.writerow(['Alice', 25])

# DictWriter
with open('out.csv', 'w', newline='') as f:
    fieldnames = ['name', 'age']
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerow({'name': 'Bob', 'age': 30})
```

**JSON Module**:
```python
import json

# Serialize (Python → JSON)
data = {'name': 'Alice', 'age': 25}
json_str = json.dumps(data, indent=2)  # pretty print
json_str = json.dumps(data, ensure_ascii=False)  # support unicode

# Deserialize (JSON → Python)
data = json.loads(json_str)

# File I/O
with open('data.json', 'w') as f:
    json.dump(data, f, indent=2)

with open('data.json') as f:
    data = json.load(f)
```

### Detail Teknis
- CSV: `newline=''` penting saat open file untuk writing (hindari extra newlines di Windows)
- `csv.reader` delimiter default ','; bisa `delimiter=';'` atau `'\t'`
- `csv.QUOTE_MINIMAL`, `QUOTE_ALL`, `QUOTE_NONNUMERIC`, `QUOTE_NONE` untuk quoting
- JSON: `default` parameter untuk serialize objek non-standard (contoh: datetime → str)
- `json.load()`/`dump()` bisa dengan file object atau `fp`
- JSON types: object (dict), array (list), string, number, true/false/null
- `sort_keys=True` untuk urutan key terurut saat dump
- `skipkeys=True` untuk abaikan keys yang non-serializable (tapi lebih baik handle)
