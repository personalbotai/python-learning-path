## Materi

### Konsep Dasar

**Logging** adalah cara mencatat events yang terjadi selama program berjalan. Lebih baik dari `print()` karena:
- Dapat di-level (DEBUG, INFO, WARNING, ERROR, CRITICAL)
- Dapat di-output ke berbagai destinations (console, file, email, dll)
- Dapat dikonfigurasi tanpa mengubah kode
- Bermanfaat untuk production troubleshooting

**Debugging** adalah proses menemukan dan memperbaiki bugs.

### Detail Teknis

**1. Basic Logging**

```python
import logging

# Basic configuration (once, di awal program)
logging.basicConfig(level=logging.INFO, format="%(asctime)s - %(levelname)s - %(message)s")

# Logging dengan level
logging.debug("Debug message (detailed)")      # tidak shown jika level INFO
logging.info("Info message (normal)")         # shown
logging.warning("Warning message")            # shown
logging.error("Error message")                # shown
logging.critical("Critical error!")           # shown

# Logging dengan exception
try:
    1 / 0
except ZeroDivisionError as e:
    logging.error("Division failed", exc_info=True)  # log traceback
```

**2. Log Levels Hierarchy**
```
DEBUG < INFO < WARNING < ERROR < CRITICAL
```
- `DEBUG`: Detailed diagnostic info (development)
- `INFO`: Confirmation things working as expected
- `WARNING`: Something unexpected, but program continues
- `ERROR`: Serious problem, some functionality broken
- `CRITICAL`: Fatal error, program may abort

**3. Advanced Configuration**

```python
import logging

# Create logger
logger = logging.getLogger(__name__)  # __name__ = module name

# Set level
logger.setLevel(logging.DEBUG)

# Create handlers
console_handler = logging.StreamHandler()  # stdout
file_handler = logging.FileHandler("app.log")

# Set levels per handler
console_handler.setLevel(logging.INFO)   # console hanya INFO+
file_handler.setLevel(logging.DEBUG)     # file log semua

# Create formatter
formatter = logging.Formatter(
    "%(asctime)s - %(name)s - %(levelname)s - %(message)s",
    datefmt="%Y-%m-%d %H:%M:%S"
)
console_handler.setFormatter(formatter)
file_handler.setFormatter(formatter)

# Add handlers to logger
logger.addHandler(console_handler)
logger.addHandler(file_handler)

# Use logger
logger.debug("Debug message")  # ke file saja
logger.info("Info message")    # ke console dan file
```

**4. Rotating File Handler**

```python
from logging.handlers import RotatingFileHandler, TimedRotatingFileHandler

# Rotate by size
handler = RotatingFileHandler(
    "app.log",
    maxBytes=1024*1024,  # 1MB
    backupCount=5  # keep 5 backup files
)

# Rotate by time (midnight)
handler = TimedRotatingFileHandler(
    "app.log",
    when="midnight",  # or 'S', 'M', 'H', 'D'
    interval=1,
    backupCount=7  # keep 7 days
)
```

**5. JSON Logging (structured)**

```python
import json
import logging

class JSONFormatter(logging.Formatter):
    def format(self, record):
        log_record = {
            "timestamp": self.formatTime(record),
            "level": record.levelname,
            "message": record.getMessage(),
            "module": record.module,
            "funcName": record.funcName,
        }
        if record.exc_info:
            log_record["exception"] = self.formatException(record.exc_info)
        return json.dumps(log_record)

handler = logging.FileHandler("app.json.log")
handler.setFormatter(JSONFormatter())
logger = logging.getLogger(__name__)
logger.addHandler(handler)
logger.setLevel(logging.INFO)

logger.info("User login", extra={"user_id": 123, "ip": "127.0.0.1"})
# {"timestamp": "...", "level": "INFO", "message": "User login", ...}
```

**6. Debugging with pdb**

```python
import pdb

def buggy_function(x):
    result = 0
    for i in range(x):
        pdb.set_trace()  # breakpoint
        result += i
    return result

buggy_function(5)

# Commands pdb:
# n (next) - execute current line, go to next
# s (step) - step into function
# c (continue) - continue until next breakpoint
# p <expr> - print expression
# l (list) - show code around current line
# b <lineno> - set breakpoint
# q (quit) - exit debugger
```

**7. Using `breakpoint()` (Python 3.7+)**

```python
def calculate(a, b):
    result = a * b
    breakpoint()  # calls sys.breakpointhook (default: pdb.set_trace())
    return result

calculate(5, 3)
```

**8. Logging Best Practices**

```python
# GOOD: logger at module level
logger = logging.getLogger(__name__)

def process_data(data):
    logger.debug("Processing data: %s", data)  # lazy formatting
    try:
        result = heavy_computation(data)
        logger.info("Processed successfully: %d items", len(data))
        return result
    except Exception as e:
        logger.error("Failed to process data", exc_info=True)  # log traceback
        raise

# BAD: using print
def process_data(data):
    print(f"Processing {data}")  # cannot disable in production
```

**9. Structured Logging with Extra**

```python
logger = logging.getLogger(__name__)

# Add contextual information
extra = {"user_id": 123, "request_id": "abc-123"}
logger.info("User logged in", extra=extra)

# Custom filter to add fields
class RequestFilter(logging.Filter):
    def filter(self, record):
        record.request_id = get_current_request_id()  # from context
        return True

logger.addFilter(RequestFilter())
```

**10. Exception Logging**

```python
try:
    do_something()
except Exception:
    logger.exception("An error occurred")  # automatically includes traceback
    # Equivalent to:
    # logger.error("An error occurred", exc_info=True)
```

### Soal 2: Debugging Exercise

Buat fungsi dengan bug, lalu debug menggunakan `pdb`:

```python
# broken_function.py
def process_transactions(transactions):
    """
    transactions: list of dict {'amount': float, 'type': 'credit'|'debit'}
    Return net balance (credit - debit)
    """
    balance = 0
    for txn in transactions:
        if txn["type"] == "credit":
            balance += txn["amount"]
        elif txn["type"] == "debit":
            balance -= txn["amount"]
    return balance

def main():
    txns = [
        {"amount": 100, "type": "credit"},
        {"amount": 50, "type": "debit"},
        {"amount": 25, "type": "debit"},
        {"amount": 200, "type": "credit"}
    ]
    result = process_transactions(txns)
    print(f"Net balance: {result}")  # Seharusnya: 225

if __name__ == "__main__":
    main()
```
