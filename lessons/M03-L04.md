## Tujuan Pembelajaran
Create dictionaries, access values, iterate over keys/values/items, and common methods.

## Materi

### Konsep Dasar
Dictionary adalah mapping type yang menyimpan data dalam pasangan key-value. Key harus hashable (immutable) dan unik; value bisa apa pun.

**Creating Dictionaries**:
- Literal: `d = {'a': 1, 'b': 2}`
- Constructor: `d = dict(a=1, b=2)` atau `dict([('a',1), ('b',2)])`
- Empty dict: `{}` atau `dict()`

**Accessing Values**:
- `d[key]`: mengembalikan value, KeyError jika key tidak ada
- `d.get(key, default=None)`: mengembalikan value atau default jika key tidak ada
- `d.setdefault(key, default)`: jika key ada, kembalikan value; jika tidak, set d[key]=default dan kembalikan default

**Iteration**:
- `for key in d:` atau `for key in d.keys():`
- `for value in d.values():`
- `for key, value in d.items():`

### Basic Dict Methods
- `keys()`: view semua keys
- `values()`: view semua values
- `items()`: view semua (key, value) pairs
- `get(key, default)`: safe access
- `pop(key, default)`: hapus key dan kembalikan value
- `popitem()`: hapus dan kembalikan (key, value) terakhir (LIFO) (3.7+)
- `clear()`: hapus semua
- `update(other)`: merge dictionary lain
- `copy()`: shallow copy

### Detail Teknis
- Dictionary mutable: bisa tambah, ubah, hapus pasangan
- Keys harus hashable (str, int, tuple, dll); values bebas
- Python 3.7+: dict preserves insertion order (sebelumnya tidak dijamin)
- `in` operator cek key: `key in d` (O(1) average)
- View objects (keys(), values(), items()) dinamis: perubahan dict langsung terlihat
- `dict.fromkeys(iterable, value)` untuk membuat dict dengan value default
