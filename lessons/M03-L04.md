## Materi

### Konsep Dasar

Dictionary adalah mapping type yang menyimpan data dalam **key-value pairs**. Ditulis dengan `{key: value}`. Key harus **hashable** (immutable) dan unique.

**Membuat dictionary:**
```python
# Dictionary kosong
empty_dict = {}

# Dictionary dengan elemen
person = {
    "nama": "Alice",
    "usia": 25,
    "kota": "Jakarta"
}

# Alternate construction
person = dict(nama="Alice", usia=25, kota="Jakarta")
```

**Akses elemen:**
```python
person = {"nama": "Alice", "usia": 25, "kota": "Jakarta"}

# Access by key
print(person["nama"])   # "Alice"
print(person["usia"])   # 25

# Get dengan default (hindari KeyError)
print(person.get("nama"))          # "Alice"
print(person.get("pekerjaan"))     # None (default)
print(person.get("pekerjaan", "Tidak ada"))  # "Tidak ada"
```

**Update dan tambah:**
```python
person = {"nama": "Alice", "usia": 25}

# Update existing
person["usia"] = 26

# Add new key
person["pekerjaan"] = "Engineer"

# Update multiple
person.update({"kota": "Bandung", "status": "single"})
```

### Detail Teknis

**1. Dictionary Methods Utama**
```python
d = {"a": 1, "b": 2, "c": 3}

# Keys, Values, Items
keys = d.keys()        # dict_keys(['a', 'b', 'c'])
values = d.values()    # dict_values([1, 2, 3])
items = d.items()      # dict_items([('a',1), ('b',2), ('c',3)])

# Get (dengan default safe)
value = d.get("d", 0)  # 0 jika key tidak ada

# Pop (hapus dan return value)
val = d.pop("a")       # 1, d: {'b':2, 'c':3}
val = d.pop("z", 0)    # 0 jika tidak ada

# Popitem (hapus dan return last inserted pair - Python 3.7+)
key, val = d.popitem()  # ('c', 3) typically

# Clear
d.clear()  # {}

# Setdefault (insert default jika tidak ada)
d.setdefault("d", 0)  # d['d'] = 0 jika 'd' tidak ada, return value
```

**2. Iterasi Dictionary**
```python
d = {"a": 1, "b": 2, "c": 3}

# Iterate keys (default)
for key in d:
    print(key)

# Iterate keys explicitly
for key in d.keys():
    print(key)

# Iterate values
for value in d.values():
    print(value)

# Iterate key-value pairs
for key, value in d.items():
    print(f"{key}: {value}")
```

**3. Dictionary Comprehensions**
```python
# Buat dictionary dari list
numbers = [1, 2, 3, 4, 5]
squares = {n: n**2 for n in numbers}
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Dengan kondisi
even_squares = {n: n**2 for n in numbers if n % 2 == 0}
# {2: 4, 4: 16}

# Transformasi existing dict
d = {"a": 1, "b": 2, "c": 3}
upper_d = {k.upper(): v*10 for k, v in d.items()}
# {'A': 10, 'B': 20, 'C': 30}
```

**4. Merging Dictionaries**
```python
# Python 3.5+ (unpacking)
d1 = {"a": 1, "b": 2}
d2 = {"b": 3, "c": 4}
merged = {**d1, **d2}  # {'a': 1, 'b': 3, 'c': 4} - d2 overwrite d1

# Python 3.9+ (| operator)
merged = d1 | d2  # sama seperti di atas

# update() method (in-place)
d1.update(d2)  # d1 modified
```

**5. Hashable Keys**
```python
# Bisa (immutable):
d = {
    42: "integer",           # int
    3.14: "float",           # float (hashable tapi hati-hati dengan precision)
    "key": "string",         # str
    (1, 2): "tuple",         # tuple (jika elemen immutable)
    frozenset([1,2]): "frozenset"
}

# Tidak bisa (mutable):
# d[[1,2]] = "list"      # TypeError: list unhashable
# d[{"a":1}] = "dict"    # TypeError: dict unhashable
```

**6. Nested Dictionaries**
```python
users = {
    "alice": {
        "email": "alice@example.com",
        "age": 25,
        "active": True
    },
    "bob": {
        "email": "bob@example.com",
        "age": 30,
        "active": False
    }
}

print(users["alice"]["email"])  # alice@example.com
```

**7. Dictionary from Two Lists**
```python
keys = ["a", "b", "c"]
values = [1, 2, 3]

d = dict(zip(keys, values))  # {'a': 1, 'b': 2, 'c': 3}
```

**8. Default Dictionary (collections.defaultdict)**
```python
from collections import defaultdict

# Otomatis initialize value untuk key baru
dd = defaultdict(int)      # default 0
dd["a"] += 1  # dd["a"] awalnya 0, jadi jadi 1

dd_list = defaultdict(list)
dd_list["numbers"].append(1)  # otomatis buat list kosong dulu
```
