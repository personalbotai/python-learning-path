## Materi

### Konsep Dasar

String slicing memungkinkan mengambil substring dengan sintaks `string[start:stop:step]`. Karena string immutable, semua "modifikasi" membuat string baru.

**Slicing advanced:**
```python
text = "Hello, World!"

# Negative indices
print(text[-6:-1])   # "World" (dari -6 ke -2)
print(text[-5:])     # "World!" (dari -5 sampai akhir)

# Step (negative untuk reverse)
print(text[::-1])    # "!dlroW ,olleH" (full reverse)
print(text[::2])     # "Hlo ol!" (every 2nd char)
print(text[1:10:3])  # "elW" (start=1, stop=10, step=3)

# Out of range safe
print(text[0:100])   # "Hello, World!" (stop dibatasi len)
```

### Detail Teknis

**1. String Methods untuk Manipulation**

```python
s = "  Python Programming 101  "

# Strip variants
clean = s.strip()           # "Python Programming 101"
clean = s.lstrip()          # "Python Programming 101  "
clean = s.rstrip()          # "  Python Programming 101"

# Replace (all occurrences)
s2 = s.replace(" ", "_")   # "__Python_Programming_101__"
s2 = s.replace(" ", "", 2) # hapus 2 spasi pertama saja

# Split (by delimiter)
words = s.split()          # ['Python', 'Programming', '101'] (by whitespace)
words = s.split('g')       # ['  Python Pro', 'ramming 101  ']
parts = s.split(' ', 2)    # ['', '', 'Python Programming 101  '] (maxsplit=2)

# Join (dari list ke string)
words = ['Python', 'Programming', '101']
joined = " ".join(words)   # "Python Programming 101"
joined = "-".join(words)   # "Python-Programming-101"
joined = "".join(words)    # "PythonProgramming101"

# Partition (split menjadi 3 bagian)
before, sep, after = "hello world".partition(" ")
# before='hello', sep=' ', after='world'
# Always returns 3-tuple, tidak error jika separator tidak ada
before, sep, after = "hello".partition(" ")
# before='hello', sep='', after=''

# Rpartition (dari kanan)
before, sep, after = "a.b.c.d".rpartition(".")
# before='a.b.c', sep='.', after='d'

# Center, Ljust, Rjust (padding)
s = "Hello"
print(s.center(10))      # "  Hello   " (center, width=10)
print(s.ljust(10, '-'))  # "Hello-----" (left align, fill='-')
print(s.rjust(10, '*'))  # "*****Hello" (right align)

# Zfill (pad with zeros, untuk numbers)
num = "42"
print(num.zfill(5))      # "00042"
```

**2. Case Conversion**
```python
s = "HeLLo WoRLd!"

print(s.lower())         # "hello world!"
print(s.upper())         # "HELLO WORLD!"
print(s.capitalize())    # "Hello world!" (hanya first char)
print(s.title())         # "Hello World!" (setiap kata)
print(s.swapcase())      # "HEllO wOrlD!" (swap case)
```

**3. Checking Content**
```python
s = "Python3.9"

print(s.startswith("Py"))    # True
print(s.endswith(".9"))      # True
print(s.isalpha())           # False (ada angka & dot)
print(s.isdigit())           # False
print(s.isalnum())           # False (ada dot)
print(s.isspace())           # False
print(s.istitle())           # False

# Number checks
"123".isdigit()      # True
"-123".islower()    # False (tidak untuk angka)
"123.45".replace('.','').isdigit()  # True (cek numeric)

# Alpha checks
"Hello".isalpha()    # True
"Hello World".isalpha()  # False (ada space)
```

**4. Find & Count**
```python
s = "abracadabra"

# Find (return index pertama, -1 jika tidak ada)
idx = s.find("a")        # 0
idx = s.find("a", 1)     # 3 (cari dari index 1)
idx = s.find("z")        # -1 (not found)

# Rfind (dari kanan)
idx = s.rfind("a")       # 10 (last occurrence)

# Index (sama seperti find tapi raise ValueError jika tidak ada)
idx = s.index("a")       # 0
# idx = s.index("z")    # ValueError!

# Count
count = s.count("a")     # 5
count = s.count("ab")    # 2
```

**5. Split Lines**
```python
multiline = """Line 1
Line 2
Line 3"""

lines = multiline.splitlines()  # ['Line 1', 'Line 2', 'Line 3']
lines = multiline.split('\n')   # sama
```

**6. Translate & Maketrans (advanced)**
```python
# Character mapping translation
s = "hello world"
table = str.maketrans("el", "EL")  # mapping e->E, l->L
translated = s.translate(table)    # "hELLo worLd"

# Hapus characters
table = str.maketrans("", "", "aeiou")  # mapping kosong, hapus aeiou
result = "hello".translate(table)        # "hll"
```
