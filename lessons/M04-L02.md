## Tujuan Pembelajaran
Master flexible function signatures, unpacking arguments, and variable-length parameters.

## Materi

### Konsep Dasar
Python mendukung berbagai cara passing arguments ke fungsi:

**Positional Arguments**: Urutan penting.
```python
def f(a, b):
    return a + b
f(1, 2)  # a=1, b=2
```

**Keyword Arguments**: Urutan tidak penting, gunakan nama parameter.
```python
f(b=2, a=1)  # sama dengan f(1,2)
```

**Default Arguments**: Nilai default jika tidak diberikan.
```python
def f(a, b=10):
    return a + b
f(5)  # b=10
```

**Variable-Length Arguments**:
- `*args`: tuple dari positional arguments yang tidak disebutkan
- `**kwargs`: dict dari keyword arguments yang tidak disebutkan

```python
def f(*args, **kwargs):
    print(args)   # tuple
    print(kwargs) # dict
f(1, 2, x=3, y=4)  # args=(1,2), kwargs={'x':3,'y':4}
```

### Unpacking Arguments
- `*iterable` dalam call: unpack iterable menjadi positional arguments
- `**dict` dalam call: unpack dict menjadi keyword arguments

```python
def f(a, b, c):
    return a + b + c
args = (1, 2, 3)
f(*args)  # unpack tuple → f(1,2,3)

kwargs = {'a':1, 'b':2, 'c':3}
f(**kwargs)  # unpack dict → f(a=1,b=2,c=3)
```

### Detail Teknis
- Default arguments dievaluasi sekali saat definisi (hati-hati dengan mutable default seperti `def f(x=[])`)
- `*args` dan `**kwargs` adalah konvensi nama, bisa apa saja (misal `*numbers`, `**options`)
- Bisa campur: `def f(pos1, pos2, /, standard, *, kwonly1, kwonly2)`
  - `/` : parameters sebelum ini positional-only (Python 3.8+)
  - `*` : parameters setelah ini keyword-only
- `*args` bisa dikombinasi dengan `args` biasa: `def f(a, *args, b)`
- `**kwargs` selalu terakhir

## Latihan

### Soal 1
Buat fungsi `konversi_uang(amount, from_currency, to_currency)` yang menerima jumlah uang, mata uang asal, dan mata uang tujuan. Gunakan default arguments untuk `from_currency='IDR'` dan `to_currency='USD'`. Gunakan dictionary kurs (simulasi) untuk konversi. Contoh kurs: USD=15000, EUR=16000, JPY=110. Jika mata uang tidak dikenal, kembalikan None.

```python
def latihan_1(amount, from_currency='IDR', to_currency='USD'):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    print(latihan_1(15000))  # 1 USD
    print(latihan_1(100, 'USD', 'EUR'))  # approx 6.25
```

**Expected Output**:
```
1.0
6.25
```

### Soal 2
Buat fungsi `print_all(*args, **kwargs)` yang mencetak semua positional arguments dan keyword arguments dengan format:
- Positional: "Positional: (arg1, arg2, ...)"
- Keyword: "Keyword: {'key1': val1, 'key2': val2, ...}"

```python
def latihan_2(*args, **kwargs):
    # Tulis kode kamu di sini
    pass

if __name__ == "__main__":
    print_all(1, 2, 3, name='Alice', age=25)
```

**Expected Output**:
```
Positional: (1, 2, 3)
Keyword: {'name': 'Alice', 'age': 25}
```
