## Materi

### Konsep Dasar

**Type Hints** (Python 3.5+) adalah cara menambahkan type annotations ke kode Python. Python tetap dynamically typed (runtime tidak enforce), tapi type hints membantu:
- Documentation: jelas what types expected
- IDE support: autocomplete, error detection
- Static analysis: mypy, pyright bisa detect type errors sebelum runtime
- Better code design: encourages thinking about types

**Basic syntax:**
```python
def greet(name: str) -> str:
    return f"Hello, {name}"

# Variable annotations
x: int = 10
names: list[str] = ["Alice", "Bob"]
```

### Detail Teknis

**1. Basic Type Hints**

```python
# Function parameters and return type
def add(a: int, b: int) -> int:
    return a + b

# Variables
count: int = 0
name: str = "Alice"
is_active: bool = True
price: float = 19.99

# Collections
from typing import List, Dict, Tuple, Set

numbers: List[int] = [1, 2, 3]
person: Dict[str, Any] = {"name": "Alice", "age": 25}  # Any from typing
point: Tuple[float, float] = (1.5, 2.5)
unique_names: Set[str] = {"Alice", "Bob"}

# Union types (multiple possible types)
from typing import Union
def process(value: Union[int, str]) -> str:
    return str(value)

# Python 3.10+ bisa gunakan | operator
def process(value: int | str) -> str:  # same as Union[int, str]
    return str(value)
```

**2. Optional & None**

```python
from typing import Optional

def find_user(user_id: int) -> Optional[str]:
    if user_id == 1:
        return "Alice"
    return None  # OK, returns str or None

# Equivalent with Union
def find_user(user_id: int) -> Union[str, None]:
    ...

# Python 3.10+
def find_user(user_id: int) -> str | None:
    ...
```

**3. Generic Types (List, Dict, dll)**

```python
# Python 3.9+ built-in generics (no need to import List, Dict dari typing)
def process_items(items: list[str]) -> None:  # list of strings
    for item in items:
        print(item.upper())

# Python 3.8 dan bawah: from typing import List
def process_items(items: List[str]) -> None:
    ...

# Dict dengan tipe key-value
def count_chars(text: str) -> Dict[str, int]:
    counts: Dict[str, int] = {}
    for char in text:
        counts[char] = counts.get(char, 0) + 1
    return counts
```

**4. Callable (Functions as Arguments)**

```python
from typing import Callable

def apply(func: Callable[[int, int], int], a: int, b: int) -> int:
    return func(a, b)

def multiply(x: int, y: int) -> int:
    return x * y

result = apply(multiply, 3, 4)  # 12

# Callable tanpa parameter
def run_callback(cb: Callable[[], None]) -> None:
    cb()

def hello():
    print("Hello")

run_callback(hello)
```

**5. Custom Types & Type Aliases**

```python
from typing import TypeAlias

# Type alias
UserId = int
UserName = str
Coordinates = Tuple[float, float]

def get_user(user_id: UserId) -> UserName:
    return "Alice"

# Complex type alias
Vector: TypeAlias = List[float]
Matrix: TypeAlias = List[Vector]  # List of lists

def dot_product(v1: Vector, v2: Vector) -> float:
    return sum(a * b for a, b in zip(v1, v2))
```

**6. TypedDict (Structured Dict)**

```python
from typing import TypedDict

class Person(TypedDict):
    name: str
    age: int
    city: str

def print_person(person: Person) -> None:
    print(f"{person['name']} is {person['age']} years old")

alice: Person = {"name": "Alice", "age": 25, "city": "Jakarta"}
print_person(alice)

# Total=False (optional keys)
class Movie(TypedDict, total=False):
    title: str
    director: str
    year: int

movie: Movie = {"title": "Inception"}  # OK, other keys optional
```

**7. Literal Types**

```python
from typing import Literal

Mode = Literal["read", "write", "append"]

def open_file(mode: Mode) -> None:
    if mode not in ("read", "write", "append"):
        raise ValueError("Invalid mode")
    # ...

open_file("read")   # OK
open_file("text")   # mypy error: invalid literal
```

**8. Final & LiteralString (Python 3.8+)**

```python
from typing import final, LiteralString

@final
class Base:
    pass

# Subclassing Base akan error di mypy

MAX_RETRIES: Final = 3  # constant

def format_query(sql: LiteralString) -> str:
    # Hanya terima string literal, bukan variabel (SQL injection prevention)
    return sql.upper()
```

**9. Generic Classes**

```python
from typing import TypeVar, Generic

T = TypeVar('T')  # Type variable

class Box(Generic[T]):
    def __init__(self, value: T):
        self.value = value
    
    def get(self) -> T:
        return self.value
    
    def set(self, value: T) -> None:
        self.value = value

int_box = Box[int](42)
str_box = Box[str]("hello")

# Constraints
Numeric = TypeVar('Numeric', int, float)

def add_numbers(a: Numeric, b: Numeric) -> Numeric:
    return a + b
```

**10. Protocols (Structural Subtyping)**

```python
from typing import Protocol

class Drawable(Protocol):
    def draw(self) -> None: ...

class Circle:
    def draw(self) -> None:
        print("Drawing circle")

class Square:
    def draw(self) -> None:
        print("Drawing square")

def render(obj: Drawable) -> None:  # Accepts anything with draw() method
    obj.draw()

render(Circle())  # OK
render(Square())  # OK
# Duck typing at type-check time
```

**11. Annotating Generators & Iterators**

```python
from typing import Iterator, Generator

def count_to(n: int) -> Iterator[int]:
    for i in range(1, n+1):
        yield i

def process_stream() -> Generator[str, None, None]:
    # yields str, accepts None (send), returns None
    yield "start"
    yield "processing"
    yield "end"
```

**12. Overloads (Multiple Signatures)**

```python
from typing import overload

@overload
def process(value: int) -> str: ...
@overload
def process(value: str) -> int: ...

def process(value):
    if isinstance(value, int):
        return str(value)
    return len(value)
```

### Soal 1: Type-Annotated Calculator

Buat kalkulator dengan type hints lengkap:

```python
from typing import Union, List, Tuple

def calculate(
    a: Union[int, float],
    b: Union[int, float],
    operation: Literal["add", "subtract", "multiply", "divide"]
) -> Union[float, None]:
    """Perform calculation, return result or None if division by zero."""
    pass

def batch_calculate(
    pairs: List[Tuple[Union[int, float], Union[int, float]]],
    operation: Literal["add", "multiply"]
) -> List[Union[int, float]]:
    """Calculate for list of (a,b) pairs."""
    pass

def main():
    # Test calculate
    result = calculate(10, 2, "add")  # 12
    result = calculate(10, 0, "divide")  # None
    
    # Test batch_calculate
    pairs = [(1, 2), (3, 4), (5, 6)]
    results = batch_calculate(pairs, "multiply")  # [2, 12, 30]
```

### Soal 2: TypedDict untuk JSON Config

Buat config system dengan TypedDict:

```python
from typing import TypedDict, NotRequired
from pathlib import Path

class DatabaseConfig(TypedDict):
    host: str
    port: int
    username: str
    password: str
    database: str

class LoggingConfig(TypedDict, total=False):
    level: str
    file: str
    max_size: int

class AppConfig(TypedDict):
    name: str
    debug: bool
    database: DatabaseConfig
    logging: NotRequired[LoggingConfig]  # optional

def load_config(path: Path) -> AppConfig:
    """Load JSON config and return typed dict"""
    import json
    with open(path) as f:
        data = json.load(f)
    # mypy akan memverifikasi struktur data sesuai AppConfig
    return data

def validate_config(config: AppConfig) -> List[str]:
    """Validate config, return list of errors"""
    errors = []
    db = config["database"]
    if not (1 <= db["port"] <= 65535):
        errors.append("Invalid port")
    return errors
```

### Soal 3: Generic Container Class

Buat generic container `Box[T]` yang bisa menyimpan value of any type, dengan methods:
- `get() -> T`
- `set(value: T) -> None`
- `map[U](func: Callable[[T], U]) -> Box[U]` - apply function ke value, return new Box

```python
from typing import Generic, TypeVar, Callable

T = TypeVar('T')
U = TypeVar('U')

class Box(Generic[T]):
    def __init__(self, value: T):
        self._value = value
    
    def get(self) -> T:
        return self._value
    
    def set(self, value: T) -> None:
        self._value = value
    
    def map(self, func: Callable[[T], U]) -> Box[U]:
        return Box(func(self._value))
    
    def __repr__(self) -> str:
        return f"Box({self._value!r})"

def main():
    box_int = Box(42)
    print(box_int.get())  # 42
    
    box_str = box_int.map(str)
    print(box_str)  # Box('42')
    
    box_list = Box([1, 2, 3])
    box_len = box_list.map(len)
    print(box_len)  # Box(3)
```
