# Generators and yield
**ID**: `M08-L02`  
**Type**: lesson  
**Duration**: 30-60 menit  
**Tags**: python, tutorial

## Tujuan Pembelajaran
Write generator functions, use yield for lazy evaluation, understand generator expressions.

## Materi

### Konsep Dasar
**Generator**: Type iterator yang dibuat dengan fungsi yang menggunakan `yield`. Lebih sederhana daripada custom iterator class.

```python
def count_up_to(n):
    i = 0
    while i < n:
        yield i
        i += 1

for x in count_up_to(5):
    print(x)  # 0 1 2 3 4
```

**Generator Expression**: Mirip list comprehension tapi dengan parentheses, menghasilkan generator (lazy).
```python
gen = (x*x for x in range(10))
```

### Lazy Evaluation
Generator menghasilkan nilai satu per satu saat diminta (dengan `next()` atau `for`). Hemat memory untuk large sequences atau infinite sequences.

**Generator State**: Generator mempertahankan state antar `yield`. Setelah selesai, `StopIteration` diraise. Tidak bisa restart; harus buat baru.

### Advanced Generator Features
- `yield` bisa mengirim nilai keluar dan menerima nilai masuk via `.send(value)`
- `yield from` untuk delegasi ke sub-generator
- `return` dalam generator mengembalikan nilai (Python 3.3+), bisa di-catch di `StopIteration.value`
- `next(gen, default)` untuk default value saat habis
- Generator adalah iterator, bisa digunakan di `for`, `list()`, `tuple()`, dll.

### Detail Teknis
- Generator function mengembalikan `generator object` (jenis iterator)
- `gen.__next__()` sama dengan `next(gen)`
- `send()` hanya bekerja jika generator sedang paused di `yield`; pertama kali harus `next(gen)` atau `send(None)`
- `throw(exc_type, exc_val, exc_tb)` untuk raise exception di dalam generator
- `close()` untuk menutup generator prematurely
- Generator expression sintaks: `(expr for item in iterable if condition)`
- Generator lebih慢 daripada list untuk small sequences karena overhead, tapi lebih baik untuk large data
