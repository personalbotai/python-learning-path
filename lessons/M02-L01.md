## Materi

### Konsep Dasar

Loop `for` digunakan untuk melakukan iterasi (perulangan) atas sequence (list, tuple, string, range, dll) atau objek yang bisa di-iterate.

**1. Loop dengan `range()`**
```python
# range(start, stop, step)
for i in range(5):           # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):       # 1, 2, 3, 4, 5
    print(i)

for i in range(1, 10, 2):   # 1, 3, 5, 7, 9
    print(i)

for i in range(10, 0, -1):  # 10, 9, 8, ..., 1
    print(i)
```

**2. Loop pada list**
```python
fruits = ["apel", "pisang", "jeruk", "mangga"]

for fruit in fruits:
    print(fruit)
# Output:
# apel
# pisang
# jeruk
# mangga
```

**3. Loop pada string**
```python
for char in "Python":
    print(char)
# Output: P, y, t, h, o, n (setiap karakter)
```

**4. Loop pada dictionary**
```python
person = {"nama": "Alice", "usia": 25, "kota": "Jakarta"}

# Iterasi keys (default)
for key in person:
    print(key)  # nama, usia, kota

# Iterasi values
for value in person.values():
    print(value)

# Iterasi key-value pairs
for key, value in person.items():
    print(f"{key}: {value}")
```

### Detail Teknis

**1. `enumerate()` - mendapatkan index dan value**
```python
fruits = ["apel", "pisang", "jeruk"]

# without enumerate
for i in range(len(fruits)):
    print(f"{i}: {fruits[i]}")

# with enumerate (lebih pythonic)
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# dengan start index custom
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}: {fruit}")  # 1: apel, 2: pisang, ...
```

**2. `zip()` - iterate multiple sequences sekaligus**
```python
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]
cities = ["Jakarta", "Bandung", "Surabaya"]

for name, age, city in zip(names, ages, cities):
    print(f"{name} berusia {age} tahun di {city}")
# Output:
# Alice berusia 25 tahun di Jakarta
# Bob berusia 30 tahun di Bandung
# Charlie berusia 35 tahun di Surabaya
```

**3. Nested loops (loop di dalam loop)**
```python
for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
# Output:
# i=0, j=0
# i=0, j=1
# i=1, j=0
# i=1, j=1
# i=2, j=0
# i=2, j=1
```

**4. `break` dan `continue`**
```python
# break: keluar dari loop
for i in range(10):
    if i == 5:
        break  # loop berhenti di i=5
    print(i)  # 0,1,2,3,4

# continue: skip ke iterasi berikutnya
for i in range(5):
    if i == 2:
        continue  # tidak print 2
    print(i)  # 0,1,3,4
```

**5. `else` clause pada loops**
```python
# else dieksekusi jika loop selesai tanpa break
for i in range(5):
    print(i)
else:
    print("Loop selesai!")  # akan dieksekusi

for i in range(5):
    if i == 3:
        break
else:
    print("Ini tidak akan tampil jika ada break")
```
