## Materi

### Konsep Dasar
Statement `if` digunakan untuk eksekusi bersyarat. Sintaksnya:
```python
if kondisi:
    # kode jika True
elif kondisi_lain:
    # kode jika True
else:
    # kode jika semua False
```
`elif` opsional, bisa banyak. `else` juga opsional. Kondisi berupa expression yang dievaluasi ke boolean. Python menggunakan indentasi untuk blok.

Operator perbandingan: `==`, `!=`, `<`, `>`, `<=`, `>=`. Gabungan dengan `and`, `or`, `not`. Nested if dimungkinkan tapi harus hati-hati agar tidak terlalu dalam (spaghetti code). Ternary operator: `x if condition else y`.

### Detail Teknis
- Python tidak memiliki switch statement, gunakan if-elif
- Perbandingan string dilakukan leksikografis
- Gunakan parentheses untuk memperjelas precedence
- Satu baris if: `if x > 0: print("positive")`
